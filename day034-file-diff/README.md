# Day 034: 2ファイル差分比較ツール（Python）

2つのテキストファイルを比べて、「どの行が増えた／減った／変わった」かを表示する実務ツールです（外部ライブラリなし・ネット通信なし）。
任意で、左右に並べて色分けしたHTML差分を書き出し、ブラウザで確認できます。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python diff_tool.py`（WindowsならダブルクリックでもOK）
2. 比べるファイル1（元）のパスを入れる
3. 比べるファイル2（新）のパスを入れる
4. 差分の概要（追加・削除・変わったまとまり）と、`+`/`-` 付きの差分が表示される
5. 希望すれば `差分レポート.html` を書き出し（ダブルクリックでブラウザ表示）

## 機能
- 行ごとの差分表示（`+` が追加、`-` が削除）
- 追加行数・削除行数・変わったまとまりの数を集計
- 文字コードは UTF-8 / Shift_JIS(cp932) を自動判別
- バイナリ（画像など）は、ヌル文字や制御文字の多さで見分けて対象外
- 左右並びのHTML差分を出力（中身は自己完結・ネット参照なし）
- 出力は連番で**上書きしない**（元ファイルは読むだけ）

## 使った技術
- Python（標準ライブラリ `difflib` / `re` / `pathlib` のみ・外部通信なし）

## 学び・ポイント
- 差分は `difflib` で簡単に作れる。`unified_diff` は `+`/`-` 付きの行、
  `HtmlDiff` は左右に並べた色分けHTMLを作ってくれる
- `SequenceMatcher(autojunk=False)` にすると、同じ行が多いファイルでも素直に比較できる
- `HtmlDiff` の出力は先頭にW3CのDTD URL付きDOCTYPEが入る。ブラウザは読みに行かないが、
  「ネット参照なし」を明確にするため **HTML5のDOCTYPEに置き換え**て完全自己完結にした
- 比べるのは「行の内容」。改行コード（LF/CRLF）の違いは見ないので、
  「中身が同じ」ではなく **「行の内容が同じ」** と正確に伝える

## 注意
- このツールは元ファイルを**書き換えません**（比べて表示・HTML出力のみ）。
- 改行コードの違いや末尾改行の有無は差分に含めません（行の内容で比較します）。
