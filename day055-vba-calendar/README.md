# Day 055: カレンダー自動生成（Excel VBA）

年と月を指定すると、新しいシートにその月のカレンダーを作るマクロです。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `GenerateCalendar.bas` を取り込む
2. `Alt + F8` →「GenerateCalendar」を実行
3. 年・月を入力（空Enterで今年・今月）

## 機能
- 指定した月のカレンダーを新シート（`YYYY-MM`）に生成
- タイトル（例: `2026年6月`）＋曜日見出し（日〜土）＋日付グリッド
- 日曜＝赤、土曜＝青、今日＝黄色で強調
- 列幅・罫線を自動調整（罫線は最終日の行まで）
- 入力は整数のみ受け付け（全角数字もOK・小数や記号は拒否）

## 使った技術
- Excel VBA（DateSerial / Weekday / Range.Merge / Borders）

## 学び・ポイント
- その月の末日は `Day(DateSerial(年, 月+1, 0))`（翌月の0日＝当月末日）で求められる
- 1日が何曜日かは `Weekday(日付, vbSunday)`（日曜=1〜土曜=7）で分かり、これが**開始列**になる
- 土曜（7列目）の次は翌週へ折り返す（列を1に戻して行を1つ下げる）
- `9999年12月` は翌月計算で日付があふれるので、入力上限を `9998` にして安全にする
- 生成途中で失敗したら、作りかけのシートを消す（中途半端を残さない）

## 注意
- 年は 1900〜9998、月は 1〜12 で入力してください。
- 祝日の色分けはしていません（曜日のみ）。
- マクロを含むブックは `.xlsm` 形式で保存してください。
