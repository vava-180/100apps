# Day 052: ブック内ハイパーリンク目次（Excel VBA）

ブックの中のすべてのシートへ飛べる「目次」シートを作るマクロです。目次の各行はハイパーリンクで、クリックするとそのシートのA1へ移動します。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 目次を作りたいブックを開く（`sample/目次サンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `CreateTableOfContents.bas` を取り込む
3. `Alt + F8` →「CreateTableOfContents」を実行

## 機能
- 「目次」シートを先頭に作り、各シートへの内部リンクを並べる
- すでに「目次」シートがあれば、中身を作り直して先頭へ移動
- シート名のアポストロフィ（`'`）は自動でエスケープ
- 非表示シート・グラフシートは対象外（リンク切れを防ぐ）
- 表示の微調整（列幅・選択）に失敗しても、目次作成自体は成功扱い

## 使った技術
- Excel VBA（Hyperlinks.Add の内部リンク / Worksheet.Move）

## 学び・ポイント
- ブック内リンクは `Hyperlinks.Add(Anchor, Address:="", SubAddress:="'シート名'!A1")` で作る
  （`Address` は空、行き先は `SubAddress`）
- シート名に `'` があると参照式が壊れるので、`Replace(name, "'", "''")` で**二重化**する
  （`\ / ? * [ ] :` はシート名に使えないので追加のエスケープは不要）
- 非表示シートへのリンクはクリックしても飛べず「壊れたリンク」に見えるので、目次から外す
- 「目次」という名前のシートがグラフシート等だと名前が衝突するので、事前に確認して中止する

## 注意
- 目次に並ぶのは「表示中のワークシート」だけです（非表示・グラフシートは対象外）。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/目次サンプル.xlsx` … 「4月／5月／6月／集計／メモ」の5シート。
  実行すると、先頭に「目次」ができて各シートへのリンクが並びます。
