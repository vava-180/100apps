# Day 043: 重複行のハイライト／削除（Excel VBA）

いま開いているシートの表（1行目＝見出し、2行目以降＝データ）から、**全部の列が同じ重複行**を見つけて、「色を付ける」または「削除する（1件目を残す）」マクロです。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 表のあるシートを開く（`sample/重複サンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `HighlightOrRemoveDuplicates.bas` を取り込む
   （または中身をコピーして標準モジュールに貼り付け）
3. `Alt + F8` →「HighlightOrRemoveDuplicates」を実行
4. `1`（色を付ける）か `2`（削除する）を入力
   - 削除は確認のうえで実行（元に戻せません）

## 機能
- 全列が同じ重複行を検出（比較する列は「見出し行の幅」ぶん）
- モード1：2件目以降を黄色にハイライト
- モード2：2件目以降を削除（1件目は残す）
- 大文字小文字は区別しない（`ABC` と `abc` は同じ）
- 中身が全部空の行は対象外
- エラー値（`#N/A` など）が入っていても止まらない

## 使った技術
- Excel VBA（Scripting.Dictionary / Range.Find / Union による一括削除）

## 学び・ポイント
- 「行が同じか」は、各セルの値をつないだ**キー**を作り、Dictionaryで既出か調べると速い
- キーは値の前に**「長さ:」を付けて**つなぐと、値の中に区切り文字が入っても取り違えない
- 削除は1件ずつ消すと**行番号がずれる**。`Union` でまとめてから一度に消すとずれない
- 比べるのは「セルの値」。数値の `1` と文字の `"1"` は同じ扱い（表示形式は見ない）

## 注意
- **削除モードは「行全体」を消します**。表の右側に別のデータやメモがあると一緒に消えるのでご注意ください。
- ハイライトは既存の背景色を上書きします（再実行しても前回の黄色は消えません）。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/重複サンプル.xlsx` … 「田中太郎」が3行、「佐藤花子」が2行ある名簿。
  実行すると、2件目以降（重複3行）が色付け／削除されます。
