# Day 023: PDF結合・分割ツール（Python）

PDFをまとめたり、分けたりする実務ツールです。
フェーズ2で初めて外部ライブラリ（`pypdf`）を使います（ローカル動作・ネット通信なし）。

## 必要なもの
- Python（3.8以上の目安）
- `pypdf`（PDFを読み書きするライブラリ）。1回だけ次でインストール:
  ```
  pip install pypdf
  ```
  ※ `pypdf` はネット通信をするものではなく、PDFをローカルで処理するためのライブラリです。

## 使い方
1. ターミナルで `python pdf_tool.py`（WindowsならダブルクリックでもOK）
2. PDFが入っているフォルダのパスを入れる
3. やることを選ぶ
   - 1) 結合：フォルダ内の複数PDFを1つに（`merged.pdf`）
   - 2) 分割：1つのPDFを1ページずつ（`〇〇_split` フォルダ）
   - 3) 抜き出し：`1-3,5` のように指定したページだけ取り出す（`〇〇_extract.pdf`）

## 機能
- 複数PDFの結合（名前順）
- PDFの1ページずつ分割
- ページ範囲を指定して抜き出し
- パスワード付きPDFは安全のため対象外（結合では飛ばす）
- 出力ファイルが既にあれば連番を付けて**上書きしない**

## 使った技術
- Python ＋ `pypdf`（外部通信なし）

## 学び・ポイント
- PDF操作は標準ライブラリだけでは難しいので、専用ライブラリ `pypdf` を使う
  （ネット通信をしないライブラリは、同梱して使ってよい）
- ファイルを作るツールは、**既存ファイルを上書きしない**工夫（連番）が大事
- 壊れたPDFが1つあっても全体を止めず、そのファイルだけ飛ばすと使いやすい
- ページ指定は `1-3,5` のような入力を、プログラムが扱える番号に**検証しながら**変換する

## 注意
- 結合では、このツールが作った `merged〜` や `〜_extract` のPDFは混ざらないよう除外します。
