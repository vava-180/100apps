# Day 007: CSV→表プレビュー＆印刷

CSVファイルを読み込むか貼り付けると、見やすい表で表示するブラウザツールです。検索・並べ替え・印刷ができます。

## 使い方
1. `index.html` をブラウザで開く
2. 「CSVファイルを選ぶ」で読み込む、またはCSVを直接貼り付ける（「サンプルを表示」でお試しも可）
3. 見出しをクリックすると並べ替え、検索ボックスで絞り込みができる
4. 「🖨 印刷」を押すと、ボタン類を隠した状態で表だけ印刷できる

## 機能
- CSVファイルの読み込み（UTF-8）／直接貼り付け
- ダブルクォート対応の簡易CSVパーサ（`"a,b"` のようにカンマを含むセルもOK）
- 表の中をまとめて検索（絞り込み）
- 見出しクリックで並べ替え（数字は数値順、文字は日本語順）
- 印刷用レイアウト（ボタン・入力欄を隠して表だけ印刷）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- FileReader API（ローカルのファイルを読むブラウザ標準機能。ネット通信はしません）

## 学び・ポイント
- CSVは `split(',')` だけだと「セル内のカンマ」で崩れるので、ダブルクォートを見ながら1文字ずつ処理する
- `@media print { ... }` を使うと、画面用と印刷用で見た目を変えられる
