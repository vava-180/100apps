# Day 037: 差し込みテキスト量産ツール（Python）

CSVの名簿と、ひな形テキスト（差し込み文）から、1行（1人）ごとの文章をまとめて作る実務ツールです（外部ライブラリなし・ネット通信なし）。
例: 「{氏名} 様、いつもお世話になっております。…」を、名簿の人数分つくる。

> ⚠ このツールは**文章を作るだけ**で、メール送信などのネット通信は一切しません。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. 名簿CSV（先頭行が見出し）と、ひな形テキスト（.txt）を用意する
   - ひな形の中に `{見出し名}` と書くと、その列の値に差し替わる（例: `{氏名} 様`）
2. ターミナルで `python merge_text.py`（WindowsならダブルクリックでもOK）
3. 名簿CSV → ひな形txt の順にパスを入れる
4. 出力方法を選ぶ（1=1ファイルにまとめる／2=1人1ファイル）
5. プレビュー（1件目）を確認し、`yes` で書き出し

## 機能
- CSVの各行ごとに、ひな形の `{見出し名}` を値へ差し込み
- 出力は「1ファイルにまとめる」か「1人1ファイル」を選べる
- ひな形の `{未知の項目}`（見出しに無い）を検知して確認、空欄になる行数も警告
- 見出しの重複・前後空白、空行、空のひな形も知らせる
- 1人1ファイルのファイル名は安全化（Windows禁止文字・予約名・制御文字に対応）
- 出力は連番で**上書きしない**（元のCSV・ひな形は読むだけ）

## 使った技術
- Python（標準ライブラリ `csv` / `io` / `re` / `sys` / `pathlib` のみ・外部通信なし）

## 学び・ポイント
- 差し込みは「ひな形を**1回だけ**走査して置き換える」のが安全（`re.sub`）。
  順番に `replace` すると、差し込んだ値の中の `{別項目}` まで再度差し込まれる事故が起きる
- `str.format` は使わない。ひな形に普通の波括弧があっても壊れないようにするため
- ファイル名は外部の値（名簿）から来るので、**禁止文字・予約名（CON など）・制御文字**を
  安全化しないと、その人だけ書き出しに失敗する
- プレビューは1件目だけなので、2件目以降の**空欄・列不足を件数で知らせる**と親切

## 注意
- 送信機能はありません（文面を作るだけ）。中身を確認してからご利用ください。
- ひな形の波括弧 `{…}` は差し込み専用です。
