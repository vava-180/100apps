# Day 058: 全スライドのフォント統一（PowerPoint VBA）

今開いているプレゼンの、すべてのスライドの文字を、指定したフォントにそろえるマクロです。バラバラになりがちなフォントを一気に統一できます。

## 必要なもの
- Microsoft PowerPoint（VBAが使えるデスクトップ版）

## 使い方
1. PowerPointでプレゼンを開く
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `UnifyFont.bas` を取り込む
3. `Alt + F8` →「UnifyFont」を実行
4. フォント名を入力（例: `メイリオ`）

## 機能
- 全スライドの文字を指定フォントに統一（英数字用＋日本語用の両方）
- 図形・テキストボックス・プレースホルダ・**表・グループの中**まで対象
- 変更できた数／できなかった数を表示
- 対応していない図形は飛ばして処理を続ける

## 使った技術
- PowerPoint VBA（Shapes 再帰 / Table.Cell / Font.Name・NameFarEast）

## 学び・ポイント
- グループ図形は `shp.Type = msoGroup` で見分け、`GroupItems` を**再帰**でたどると入れ子もOK
- 表は `shp.HasTable` で見分け、`Table.Cell(r,c).Shape.TextFrame` で各セルの文字を変える
- 日本語は `Font.NameFarEast` も設定しないと、英数字だけ変わって日本語が変わらないことがある
- `On Error Resume Next` で握りつぶすと「失敗も成功」に見えるので、**設定後に `Err.Number` を確認**して、
  成功した箇所だけ数える

## 注意
- 元に戻せないので、必要なら先に保存してください。
- フォント名が正しいかは確認していません（入力後、表示をご確認ください）。
- マクロを含むプレゼンは `.pptm` 形式で保存してください。
