# Day 015: ストップウォッチ＆タイマー

時間の計測（ラップ付き）と、カウントダウンタイマーができるブラウザツールです。終了時に音で知らせます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 上のタブで「ストップウォッチ」か「タイマー」を選ぶ
3. ストップウォッチ：スタート→ラップ／ストップ→リセット
4. タイマー：分・秒を入れてスタート。0になると音とともに点滅で知らせる

## 機能
- ストップウォッチ（1/100秒表示・ラップ記録）
- カウントダウンタイマー（分・秒指定・終了音）
- スタート／一時停止／再開／リセット
- タブを切り替えると、動作中のものは自動で一時停止

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- Web Audio API（ブラウザ内で「ピピピ」という終了音を生成。ネット通信はしません）

## 学び・ポイント
- `setInterval` は少しずつズレるので、経過時間は **`Date.now()` との差で毎回計算**する。
  こうすると、表示の更新がカクついても合計時間がズレない
- 「開始／一時停止／リセット」のような状態は、止めるときに**ボタンの有効・無効も合わせて切り替える**と、
  止まっているのにラップが押せる、といった取りこぼしを防げる
- 音は外部ファイルを使わず、Web Audio APIで波（オシレーター）を作って鳴らせる
