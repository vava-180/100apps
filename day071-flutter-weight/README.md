# Day 071: 体重記録（Flutter）

体重を入力して記録し、**最新の値**と**前回との差**を表示する簡単な記録アプリです。小数の入力と、前のデータと比べる処理を学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 体重（kg・小数OK）を記録（記録した日時も自動で保存）
- 最新の体重を大きく表示／前回との差（増減）を色分け表示（増=赤・減=青）
- 1件ずつ削除
- 0以下や500kg超など、ありえない値ははじく（SnackBar）

## 使った技術
- Flutter / Dart（自作クラス WeightLog・double.tryParse・DateTime）
- 追加パッケージなし

## 学び・ポイント
- 体重は小数なので `int` ではなく `double.tryParse` を使う
- 「最新」と「前回」を比べるときは、新しい順にリストへ入れておくと `[0]` と `[1]` で簡単に取り出せる
- `toStringAsFixed(1)` で小数1桁にそろえると見やすい

## 注意
- データはアプリ内（メモリ）に保持するため、**アプリを終了すると消えます**。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
