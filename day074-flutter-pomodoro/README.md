# Day 074: ポモドーロタイマー（Flutter）

「**25分作業 → 5分休憩**」を繰り返す集中タイマーです。0 になると自動で作業／休憩が切り替わります。1秒ごとのカウントダウンと、状態の切り替えを学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- スタート／一時停止／リセット
- 残り時間をリング（円形プログレス）の中に大きく表示
- 0になると自動で「作業 ⇄ 休憩」を切り替え
- 完了した作業回数をカウント

## 使った技術
- Flutter / Dart（`Timer.periodic`・`CircularProgressIndicator`・状態フラグ）
- 追加パッケージなし

## 学び・ポイント
- カウントダウンは「残り秒を1ずつ減らし、0になったら次のフェーズへ」とすると分かりやすい
- **`Timer` は使い終わったら必ず `cancel()`**（`dispose` でも止める）。`_timer == null` で「動いているか」を判定できる
- 進捗リングの `value` は 0.0〜1.0。`残り ÷ 全体` を渡すと残量バーになる

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
