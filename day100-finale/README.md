# Day 100: 総集編アプリ（チャレンジの集大成）

100日100アプリチャレンジの最終作。これまでの歩みをまとめた「集大成ダッシュボード」です。達成の演出、フェーズ別の進捗、そして全アプリからランダムに1本引く「アプリガチャ」を1ページに詰め込みました。1ファイル・オフラインで動きます。

## 使い方
1. `index.html` をブラウザで開く
2. 実績・フェーズ進捗を見る
3. 「ランダムに1本引く」で、これまでのアプリをランダムに開ける

## 機能
- 達成バッジ（100 / 100）と数字で見る実績
- フェーズ別の進捗バー（一覧データから自動計算）
- **アプリガチャ**：全アプリからランダムに1本選び、リンクで開ける
- 記録サイト・ポートフォリオ・GitHub への導線
- ふりかえりメッセージ

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- アプリ一覧データは、各 `dayXXX` フォルダから自動抽出して埋め込み

## 学び・ポイント
- アプリ一覧を JavaScript の配列として埋め込むと、フェーズ集計やガチャをデータ起点で書ける
- ランダム選択は `Math.floor(Math.random() * 配列.length)`
- 画面に文字を入れるときは `textContent` を使うと、タグ混入（XSS）を防げる
- 進捗バーは「達成数 ÷ 全体 × 100%」を `width` に入れるだけで表現できる

## 注意
- アプリ自体はネット通信しません（外部APIの呼び出しなし）。GitHubへのリンクのみ、押すと外部サイトを開きます。
- 各アプリへのリンクは、このフォルダから見て1つ上の階層を指します。
