# Day 004: 西暦↔和暦＆営業日計算

日付を選ぶと和暦に変換し、土日を除いた営業日や、日付どうしの間の日数も計算するブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. 「西暦 → 和暦」で日付を選ぶと、令和○年…の表示になる
3. 「営業日を計算」で起点の日付と日数を入れると、土日を飛ばした日付が出る
4. 「2つの日付の間の日数」で2つの日付を入れると、差の日数が出る

## 機能
- 西暦 → 和暦の変換（明治〜令和に対応、1年目は「元年」と表示）
- 営業日計算（土日を除いて「○営業日後／前」の日付を算出）
- 2つの日付の間の日数を計算

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 元号は「開始日の一覧」を新しい順に持っておき、選んだ日付が「どの開始日以降か」で判定する
- 曜日は `getDay()` が `0=日曜〜6=土曜` を返すので、`0`と`6`をスキップすれば土日を除外できる
- 日数の差は「ミリ秒の差 ÷ 86,400,000（1日のミリ秒）」で求められる

## 注意
- 営業日計算は祝日を考慮していません（土日のみ除外）。
