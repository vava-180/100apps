# Day 048: 請求書テンプレ一括出力（Excel VBA）

同じブックの中にある「請求データ」と「テンプレート」シートから、請求書を件数分まとめて作るマクロです。1件ごとにテンプレートを複製し、差し込み記号を値に置き換えて、新しいブックに出力します。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## ブックの形（2つのシート）
- **「請求データ」**：1行＝1件。1行目が見出し（例: `請求先` / `件名` / `金額`）
- **「テンプレート」**：セルに `{請求先}` `{件名}` `{金額}` `{発行日}` のような差し込み記号を書く

## 使い方
1. 上の2シートがあるブックを開く（`sample/請求書サンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `GenerateInvoices.bas` を取り込む
3. `Alt + F8` →「GenerateInvoices」を実行
4. 新しいブックに、1件＝1シートの請求書ができる（保存はご自身で）

## 機能
- 1件ごとにテンプレートを複製し、`{見出し名}`→値、`{発行日}`→今日 に差し替え
- セル全体がちょうど `{金額}` などのときは、**値の型（数値・日付）を保って**差し込み
- 差し込みは**1パス**で行い、値の中に `{別の記号}` があっても二重に差し込まない
- 差し込めなかった記号 `{…}` が残った請求書は件数で警告
- シート名は「請求先」（無ければ連番）。Excelの制限・重複は自動調整
- 空行はスキップ。元のデータは変更しない

## 使った技術
- Excel VBA（Worksheet.Copy / 文字列の1パス置換 / Range.Find）

## 学び・ポイント
- 差し込みは「ひな形を**1回だけ**走査して置換」すると、差し込んだ値の中の `{記号}` まで
  再び差し込む事故（二重差し込み）を防げる
- セル全体が1つの記号のときは、文字列ではなく**セルの値そのまま**を入れると、金額や日付の型が保てる
- 最終行は A列だけで判断せず、`Cells.Find(SearchDirection:=xlPrevious)` でシート全体から取る
- `Application.DisplayAlerts` を一時的にFalseにしたら、**必ず元の値に戻す**

## 注意
- 数式セルの中の `{…}` は差し込みません（数式を壊さないため）。
- 結果は新しいブックに出ます。必要に応じて保存してください。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/請求書サンプル.xlsx` … 「請求データ」3件＋「テンプレート」。
  実行すると、山田商事・佐藤工業・鈴木物産の3枚の請求書シートができます。
