# Day 059: 画像を一括でスライド化（PowerPoint VBA）

指定フォルダの中の画像（jpg / png / gif / bmp）を、名前順に1枚ずつスライドにするマクロです。写真をまとめてスライドに並べたいときに便利です。

## 必要なもの
- Microsoft PowerPoint（VBAが使えるデスクトップ版）

## 使い方
1. PowerPointでプレゼンを開く（無ければ新規作成してから）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `ImagesToSlides.bas` を取り込む
3. `Alt + F8` →「ImagesToSlides」を実行
4. 画像が入ったフォルダのパスを入力（`sample` フォルダで試せます）

## 機能
- フォルダ内の画像を名前順に、1画像＝1スライドで挿入（白紙レイアウト）
- 画像はスライドの9割に収まるよう、**縦横比を保って**拡大縮小し中央配置
- `*.jpg` と `*.jpeg` の二重取りを防止（一度だけ列挙して拡張子で判定）
- 壊れて読めない画像はスキップし、追加した空スライドを残さない

## 使った技術
- PowerPoint VBA（Shapes.AddPicture / LockAspectRatio / Dir）

## 学び・ポイント
- 画像は `Shapes.AddPicture(..., Width:=-1, Height:=-1)` で**元の大きさ**で入る
- `LockAspectRatio = msoTrue` にすると、幅を変えれば高さも比例して変わる（縦横比キープ）
- 「スライドに収める倍率」は、幅・高さそれぞれの比のうち**小さい方**を選ぶとはみ出さない
- `*.jpg` と `*.jpeg` を別々に集めると二重に取ることがあるので、**一度だけ列挙**して拡張子で選ぶ
- 失敗したら、先に追加した白紙スライドを消すと、空スライドが混ざらない

## 注意
- 画像はファイルに埋め込みます（`SaveWithDocument`）。リンクではありません。
- 対象はフォルダ直下の画像だけです（サブフォルダは見ません）。
- マクロを含むプレゼンは `.pptm` 形式で保存してください。

## サンプル
- `sample/` … 赤・緑・青の小さなPNG画像3枚。`sample` フォルダを指定すると、3枚のスライドができます。
