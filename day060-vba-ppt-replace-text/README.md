# Day 060: スライドの文字を一括差し替え（PowerPoint VBA）

開いているPowerPointの**全スライド**を巡って、指定した文字を別の文字へまとめて置き換えるマクロです。フォントや色などの**書式は保ったまま**、文字だけを差し替えます。社名・日付・商品名などを資料全体で一気に直したいときに便利です。

## 必要なもの
- Microsoft PowerPoint（VBAが使えるデスクトップ版）

## 使い方
1. 差し替えたいプレゼン（.pptx / .pptm）を開く
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `ReplaceTextInSlides.bas` を取り込む
3. `Alt + F8` →「ReplaceTextInSlides」を実行
4. 「検索する文字」を入力 → 「置き換える文字」を入力
5. 確認メッセージで「はい」を押すと、全スライドを置き換えて件数を表示

## 機能
- 全スライドの文字を一括で検索・置換（書式はそのまま）
- 対象：
  - 通常のテキストボックス／図形（タイトル・本文などのプレースホルダ含む）
  - **表（テーブル）のセル**の文字
  - **グループ化された図形**の中の文字（入れ子のグループも再帰で対応）
- 置き換える文字を空欄にすると「検索した文字を削除」になる
- 実行前に確認メッセージを表示／実行後に置換した件数を表示

## 対象外（触らないもの）
- ノート（発表者メモ）、SmartArt、画像の中の文字

## 使った技術
- PowerPoint VBA（Slides / Shapes / Table / TextRange.Replace）

## 学び・ポイント
- PowerPointの文字置換は `TextRange.Replace` を使うと、**部分的な書式（一部だけ色違いなど）を壊さず**に差し替えられる。`.Text` を文字列ごと書き換えると書式が飛んでしまう
- `Replace` は1回につき1か所だけ置換し、置換した範囲（`TextRange`）を返す。`Nothing` が返るまでループすれば全置換になる
- 置換文字が検索文字を含む場合（例：「A」→「AB」）に無限ループしないよう、**置き換えた文字の直後**（`found.Start + Len(置換文字) - 1`）から続きを検索する
- 図形は「グループ」「表」「普通の図形」で構造が違う。`Type = msoGroup` はグループ、`HasTable` は表。グループと表は**中に入って再帰的に**処理する
- 文字を持つかは `HasTextFrame` と `TextFrame.HasText` の2段階でチェックすると安全（画像など文字を持たない図形でエラーにならない）

## 注意
- 元に戻せないので、必要なら先に保存してから実行してください。
- 大文字・小文字は区別して検索します（`MatchCase:=True`）。
- マクロを含むブックは `.pptm` 形式で保存してください。

## 試し方（サンプルの作り方）
1. 新規プレゼンに2〜3枚スライドを作り、いくつかのスライドに「2025年」などの同じ文字を入れる
2. 表やグループ化した図形にも同じ文字を入れておく
3. マクロを実行し、「2025年」→「2026年」に置き換えると、表・グループの中もまとめて変わることを確認できます
