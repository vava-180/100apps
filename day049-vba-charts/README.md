# Day 049: グラフ一括作成（Excel VBA）

いま開いているシートの表から、**数値の列ごとにグラフを1つずつ**まとめて作るマクロです。グラフは表の下に並びます。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 表の形
- 1行目＝見出し、A列＝項目名（グラフの横軸）、B列以降＝数値（グラフの値）

## 使い方
1. 表のあるシートを開く（`sample/グラフサンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `CreateCharts.bas` を取り込む
3. `Alt + F8` →「CreateCharts」を実行
4. `1`（縦棒）／`2`（折れ線）／`3`（横棒）を選ぶ

## 機能
- 数値の列の数だけグラフを作成（横軸はA列の項目名、表題は見出し）
- 縦棒／折れ線／横棒を選べる
- グラフは表の下に2列のグリッドで並ぶ
- 再実行すると、前回このマクロが作ったグラフ（名前が `AutoChart_` で始まる）だけ作り直す
- 数値の列が無いときは、既存グラフを消す前に中止
- 表がシート下端に近くて置けない場合は事前に知らせる

## 使った技術
- Excel VBA（ChartObjects.Add / SeriesCollection.NewSeries / Range.Find）

## 学び・ポイント
- グラフは `ChartObjects.Add(Left, Top, Width, Height)` で置ける。座標は**ポイント単位**で、
  `Cells(...).Left` `.Top` と同じ座標系なので、セル基準で位置を決められる
- 系列は `NewSeries` してから `Values`／`XValues`／`Name` を入れる。
  系列名は**見出しの文字をそのまま**入れると、シート名に記号があっても壊れない
- 作り直し用に**決まった接頭辞**で名前を付けておくと、前回ぶんだけ安全に消せる
- 「消す前に、作る対象があるか確認」しておくと、誤操作で既存グラフを失わない

## 注意
- A列（項目名）は全行うめてください（データの行数はA列で数えます）。
- このマクロが消すのは `AutoChart_` で始まる名前のグラフだけです。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/グラフサンプル.xlsx` … 月別・拠点別（東京/大阪/名古屋）の売上。
  実行すると、3拠点ぶんのグラフが表の下に並びます。
