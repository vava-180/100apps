# Day 090: ミニ電卓（Flutter）

四則演算（＋ − × ÷）ができる簡単な電卓です。クリア・小数点・**ゼロ除算対策**つき。「今の入力」「前の数」「待っている計算」を状態として持つ、電卓の作り方を学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 数字・小数点の入力、四則演算（＋ − × ÷）、＝で計算
- C ですべてリセット
- 「1 ＋ 2 ＋ 3」のような連続計算に対応
- 0で割ったときは「エラー」と表示（次の入力で自動リセット）
- 結果が整数のときは小数点を付けずに表示

## 使った技術
- Flutter / Dart（状態管理：表示文字・前の数・待ち演算子・フラグ）
- 追加パッケージなし

## 学び・ポイント
- 電卓は「画面の数（_display）」「1つ前の数（_stored）」「待っている演算子（_pendingOp）」の3つを持つと作れる
- 演算子を押すたびに前の計算を確定させると、連続計算（チェーン）ができる
- 0で割る計算は `null` を返してエラー表示にすると、クラッシュを防げる
- 「次の数字でリセットするか（_startNew）」フラグが、入力の切り替えのカギ

## 注意
- 学習用のシンプルな電卓です（計算の優先順位＝×÷を先に、には対応せず、押した順に計算します）。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
