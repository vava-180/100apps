# Day 057: PowerPoint 目次スライド自動生成（PowerPoint VBA）

今開いているプレゼンの各スライドのタイトルを集めて、先頭に「目次」スライドを作るマクロです。

## 必要なもの
- Microsoft PowerPoint（VBAが使えるデスクトップ版）

## 使い方
1. PowerPointでプレゼンを開く（各スライドにタイトルが入っていると見やすい）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `CreateTocSlide.bas` を取り込む
3. `Alt + F8` →「CreateTocSlide」を実行

## 機能
- 各スライドのタイトルを集めて、先頭に「目次」スライドを生成（`2. タイトル` のように番号付き）
- すでに**このマクロが作った目次**が先頭にあれば、作り直す（タグで見分ける）
- タイトルが無いスライドは「（無題のスライド）」と表示
- 本文プレースホルダが無ければテキストボックスを作って書き込む

## 使った技術
- PowerPoint VBA（Slides.Add / Shapes.Title / Tags / Placeholders）

## 学び・ポイント
- スライドのタイトルは `Shapes.HasTitle` ＋ `Shapes.Title.TextFrame.TextRange.Text` で取れる。
  `TextFrame.HasText` を確認してから読むと、空や壊れた図形でも落ちない
- 「自分が作った目次」は `Slide.Tags` に目印を付けると、**タイトルが偶然同じ**ユーザー自作スライドを
  間違って消さずに済む
- 本文の置き場所は `PlaceholderFormat.Type` で「本文プレースホルダ」を探すと確実
- 最後の「目次へ移動」は表示状態で失敗することがあるので、失敗しても成功扱いにする

## 注意
- 目次の番号は「先頭に目次を入れた後のスライド番号」です（本文スライドは2番から）。
- ハイパーリンクは付けていません（番号とタイトルの一覧です）。
- マクロを含むプレゼンは `.pptm` 形式で保存してください。
