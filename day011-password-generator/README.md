# Day 011: パスワード生成ツール

長さと文字の種類を選んで、安全なパスワードをその場で作るブラウザツールです。入力内容はパソコンの外に送られません（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 長さ（4〜40）と、使う文字の種類（大文字・小文字・数字・記号）を選ぶ
3. 必要なら「紛らわしい文字を除く」にチェック（O 0 o, I l 1 などを使わない）
4. 自動で生成される。「コピー」でクリップボードにコピー、「⟳」で作り直し

## 機能
- 安全な乱数でパスワードを生成（暗号用の `crypto.getRandomValues` を使用）
- 文字の種類を選択（選んだ種類は必ず1文字以上ふくまれる）
- 紛らわしい文字（見間違えやすい文字）を除外するオプション
- 強度メーター（長さ×文字の種類数から「破られにくさ」を目安表示）
- ワンクリックでコピー

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- パスワードのような「予測されては困る」乱数は、`Math.random()` ではなく
  **`crypto.getRandomValues`（暗号用の安全な乱数）** を使う
- 乱数を「余り（%）」で範囲に収めるだけだと数字に偏りが出る。
  割り切れない分を捨てて引き直す（リジェクションサンプリング）と均等になる
- 「各文字種を必ず入れる」を、先頭に固定で入れるのではなく
  **全部の文字から引いて、種類がそろうまで作り直す**方式にすると、
  条件を満たすパスワードの中から公平（かたよりなく）選べる

## 注意
- 強度表示はあくまで「目安」です。実際の安全性は、使い回さない・長くする・
  サービスごとに変える、といった運用も大切です。
