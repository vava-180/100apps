# Day 089: 多言語あいさつ（Flutter）

国（言語）を選ぶと、その言語の「**こんにちは**」と読み方を大きく表示します。「選んだ項目に対応するデータを取り出す」基本の練習です。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 下部の横スクロールリストから言語（国旗）を選択
- 選んだ言語のあいさつ・読み方・言語名を大きく表示
- 日本語／英語／中国語／韓国語／フランス語／スペイン語／ドイツ語／イタリア語

## 使った技術
- Flutter / Dart（自作クラス Greeting・横スクロール `ListView.builder`・選択状態）
- 追加パッケージなし・**ネット通信なし**（あいさつはアプリ内に用意）

## 学び・ポイント
- 「選択中の番号（index）」を1つ持つだけで、対応するデータ（`kGreetings[index]`）を取り出せる
- リストを横に並べたいときは `ListView.builder` の `scrollDirection: Axis.horizontal`
- 選ばれている項目は枠線や背景色を変えると分かりやすい

## 注意
- 翻訳・読み方は学習用の簡易表記です。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
