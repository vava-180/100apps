# Day 051: 不要シート・空白行の一括削除（Excel VBA）

次のどちらかをまとめて行うマクロです。
- **空のシートを削除**（中身が何も無いワークシートを消す）
- **空白行を削除**（アクティブシートの、全部の列が空の行を消す）

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. ブック（空白行削除のときは対象シート）を開く（`sample/クリーンアップサンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `DeleteUnneeded.bas` を取り込む
3. `Alt + F8` →「DeleteUnneeded」を実行
4. `1`（空シート削除）か `2`（空白行削除）を選ぶ

## 機能
- **空シート削除**：表示中の空シートを削除。最低1枚は必ず残す。非表示シートは対象外
- **空白行削除**：全列が空の行を行全体で削除（まとめて消して行ずれを防止）
- 数式（`=""` 含む）やエラー値がある行・シートは「空ではない」とみなして消さない
- どちらも実行前に確認（元に戻せません）

## 使った技術
- Excel VBA（Worksheet.Delete / Range.Delete / Range.Find / Union）

## 学び・ポイント
- 「空のシート」は `Cells.Find(What:="*")` が `Nothing` かどうかで判定できる
- 全シートが空でも、ブックには最低1枚シートが必要。**残す枚数を計算**してから削除する
- 「見た目が空でも数式が入っている」行は消さない。`HasFormula` で守る
- 空白行は1つずつ消すと行番号がずれる。`Union` でまとめてから一度に消す
- `Find` は前回の検索書式に影響されることがあるので `SearchFormat:=False` を明示する

## 注意
- **空白行削除は「行全体」を消します**。表の右側に別データがあると一緒に消えるのでご注意ください。
- 非表示シートは空シート削除の対象外です。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/クリーンアップサンプル.xlsx` … 空白行を含む「データ」シート＋空シート2枚＋メモシート。
  モード1で空シート2枚が消え、モード2で「データ」シートの空白行が消えます。
