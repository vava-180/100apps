# Day 075: BMI計算（Flutter）

身長（cm）と体重（kg）から **BMI** を計算し、判定（低体重〜肥満）を色分けで表示します。計算式と入力チェック、結果表示の練習です。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 身長・体重を入力して BMI を計算
- 日本肥満学会の区分で判定（低体重／普通体重／肥満1〜4度）
- 結果を色分け表示（低=青・普通=緑・肥満=赤）
- 数字でない・0以下の入力ははじく（SnackBar）

## 使った技術
- Flutter / Dart（`double.tryParse`・条件分岐での判定）
- 追加パッケージなし

## 学び・ポイント
- BMI = 体重kg ÷（身長m × 身長m）。**cm を m に直す（÷100）** のがポイント
- 判定は値の範囲で分岐。`if` を小さい順に並べると読みやすい
- 結果は `double?`（未計算は null）で持ち、null のときは結果カードを出さない

## 注意
- 判定はあくまで目安です（医療的な診断ではありません）。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
