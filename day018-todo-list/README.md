# Day 018: TODOリスト（保存対応）

やることを書いて管理するブラウザツールです。閉じても消えず、このブラウザに保存されます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 入力欄にやることを書いてEnter（または「追加」）
3. 丸い印をクリックで完了／未完了を切り替え、「×」で削除
4. 「すべて／未完了／完了」で絞り込み、「完了したものを削除」でまとめて消去

## 機能
- タスクの追加・完了切替・削除
- フィルター（すべて／未完了／完了）と残り件数表示
- 完了タスクの一括削除
- `localStorage` による自動保存（閉じても残る）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- `localStorage`（ブラウザ内にデータを保存するしくみ）

## 学び・ポイント
- データ（配列）を変えたら毎回保存して、画面は配列から作り直す。
  「データ → 画面」の一方向にすると、状態の食い違いが起きにくい
- 保存データは**信用せずに検証**する。壊れた値が混ざっても落ちないよう、
  正しい形だけ通し、長すぎる文字や空は取り除いている
- 保存はプライベートモードや容量超過で失敗することがある。
  その場合も操作は止めず、「保存できません」と知らせるだけにしている
