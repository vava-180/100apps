# Day 014: 割り勘計算機

合計金額と人数から「1人いくら」を計算するブラウザツールです。端数の寄せ方も選べます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 合計金額と人数を入力する
3. 1人あたりの丸め方を選ぶ（100円単位で多め／10円単位で多め／きっちり）
4. 1人あたりの金額と、集まる合計・余りが表示される

## 機能
- 1人あたりの金額を自動計算
- 端数の丸め方を3種類から選択
- 切り上げモード：多めに集めて出る「余り」を表示
- きっちりモード：1円単位で合計がぴったり合う分け方（例：1人が334円・2人が333円）を提案

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 多めに丸めると「集めた合計 ＞ 請求額」になり、余りが出る
- きっちり割り切れないときは、`Math.floor` と「余り人数」を使うと、
  **何人が＋1円多く払えば合計がぴったり合うか**を計算できる
  （例：1000円÷3人 → 1人が334円・2人が333円＝ちょうど1000円）
- 金額・人数は整数チェックを入れて、小数や巨大値など「ありえない入力」を防いでいる
