# Day 096: ポートフォリオサイト

100日100アプリチャレンジを紹介する、1枚もの（シングルページ）の自己紹介サイトです。HTML/CSS だけで作り、外部フォントやCDNも使わずオフラインで動きます。

## 使い方
1. `index.html` をブラウザで開く
2. 上部メニューから各セクション（概要 / フェーズ / 作品 / リンク）へジャンプできる

## 機能
- ヒーロー（先頭の大きな見出し）＋ ページ内ナビ（固定ヘッダー）
- チャレンジの概要、数字で見る実績
- 5つの技術フェーズの紹介カード
- 作品ハイライト
- GitHubリポジトリへのリンク
- スマホ幅にも対応（レスポンシブ）

## 使った技術
- HTML / CSS（JavaScriptなし・外部通信なし）

## 学び・ポイント
- 主要コンテンツを `<main>`、各かたまりを `<section>`、一覧を `<ul>/<li>` にすると意味が伝わりやすい（アクセシビリティ向上）
- 文字色と背景色の**コントラスト**を十分に取ると読みやすい（薄い色は避ける）
- `a:focus-visible` でキーボード操作時の選択位置を見えるようにする
- 固定ヘッダーがあるときは `scroll-margin-top` で、ページ内リンクの見出し隠れを防ぐ
- 外部リンクは `rel="noopener noreferrer"` を付けると安全

## 注意
- 完全にオフラインで動きます（外部APIやネット通信はありません）。
- 個人情報（本名・メール等）は載せず、公開リポジトリへのリンクのみです。
