# Day 083: おみくじ（Flutter）

ボタンを押すと、運勢（大吉〜凶）と**ひとことメッセージ**をランダムに表示します。候補から1つをランダムに選ぶ基本の練習です。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- ボタンで運勢をランダムに表示（大吉／中吉／小吉／吉／末吉／凶）
- 運勢ごとに色を変え、ひとことメッセージも表示
- 何度でも引き直せる

## 使った技術
- Flutter / Dart（`Random`・自作クラス Fortune・`...`（スプレッド）での条件付き表示）
- 追加パッケージなし

## 学び・ポイント
- リストからランダムに選ぶ基本形は `list[random.nextInt(list.length)]`
- 「運勢名・色・メッセージ」を1つのクラス（Fortune）にまとめると、追加・編集がしやすい
- まだ引いていない状態は `null` で表し、`if/else` で表示を出し分ける

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
