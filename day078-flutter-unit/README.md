# Day 078: 単位変換（長さ）（Flutter）

長さの単位「mm／cm／m／km」を選んで変換します。**いったん基準の単位（m）にそろえてから変換する**という考え方を学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 値を入力し、変換元／変換先の単位をドロップダウンで選択
- 入力のたびにリアルタイム変換
- 入れ替えボタンで「元 ⇄ 先」をワンタップ交換
- 余計な小数の 0 を消して見やすく表示

## 使った技術
- Flutter / Dart（`DropdownButtonFormField`・自作クラス Unit・正規表現で末尾の0除去）
- 追加パッケージなし

## 学び・ポイント
- 単位ごとに「1単位＝何メートルか」を持っておくと、**どの単位同士でも「m を経由」**して変換でき、組み合わせが増えても式が1本で済む
- `値 × 元の係数 ÷ 先の係数` が変換の基本形
- 末尾の 0 は正規表現 `0+$` で消すと `1.50 → 1.5` になり読みやすい

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
