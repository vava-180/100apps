# Day 016: Markdownプレビュー

左にMarkdownを書くと、右にリアルタイムで表示されるブラウザツールです。HTMLとしてコピーもできます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 左の入力欄にMarkdownを書く（最初にサンプルが入っています）
3. 右にプレビューがリアルタイムで表示される
4. 「HTMLをコピー」で、変換後のHTMLをクリップボードにコピー

## 機能
- 見出し / 太字 / 斜体 / 取り消し線 / インラインコード
- 箇条書き・番号付きリスト / 引用 / 水平線 / 表 / コードブロック / リンク
- リアルタイムプレビュー、HTML出力コピー

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- Markdown→HTMLの変換を自前で実装（外部ライブラリなし）

## 学び・ポイント
- まず入力全体を **HTMLエスケープ**（`<` などを無害化）してから変換すると、
  入力に含まれるタグがそのまま実行されるのを防げる（安全になる）
- インラインコード（\` \`）の中は装飾しないので、**先に抜き出して目印に置き換え**、
  最後に戻す。目印は「@@CODE0@@」のように、ふつうの文章と衝突しない形にするのがコツ
- リンクは `href` に**URL側**を入れる（本文側ではない）。さらに http/https と # 以外は
  受け付けないことで、`javascript:` のような危ない指定を防いでいる

## 注意
- 学習用の簡易パーサです。入れ子のリストなど一部の複雑な記法には対応していません。
