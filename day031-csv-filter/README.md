# Day 031: CSV 列の抽出・絞り込みツール（Python）

1つのCSVから「必要な列だけ」を取り出し、さらに「条件に合う行だけ」に絞って、新しいCSVに書き出す実務ツールです（外部ライブラリなし・ネット通信なし）。

例: 「氏名」「金額」列だけを残し、「金額が1000以上」の行だけを出力する。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python filter_csv.py`（WindowsならダブルクリックでもOK）
2. 対象のCSVファイルのパスを入れる
3. 番号付きで列が一覧表示される
   - 残す列を番号で選ぶ（例: `1,3,5` ／ 空Enterで全部）
   - 絞り込み条件を付けるか選ぶ（任意）
     - 条件をかける列・条件（含む／等しい／含まない／数値の大小）・比べる値
4. 抽出結果のプレビューを確認
5. `yes` で `元の名前_抽出.csv` に書き出し（Excelで開きやすいUTF-8・BOM付き）

## 機能
- 列の抽出（番号で複数指定・並べ替えも可）
- 行の絞り込み（含む／等しい／含まない／数値が ＞ ≧ ＜ ≦）
- 文字条件は大文字小文字の区別を選べる
- 桁区切りカンマ（`1,234`）も数値として読む（`1,2` や `1e3`・`inf` などは数値扱いしない）
- 文字コードは UTF-8 / Shift_JIS(cp932) を自動判別し、読めた方を表示
- 出力は連番で**上書きしない**（元のCSVも書き換えない）

## 使った技術
- Python（標準ライブラリ `csv` / `io` / `math` のみ・外部通信なし）

## 学び・ポイント
- CSVは `io.StringIO(newline="")` ＋ `csv.reader(strict=True)` で読むと、
  **セル内の改行**（`"1行目\n2行目"` のような値）も壊さず読め、壊れたCSVも気づける
- 絞り込みで「**空欄**」と「**列そのものが無い行**」は意味がちがう。
  列が無い行はどの条件でも対象外にして、誤判定を防ぐ
- 「数値の大小」では `1e3` や `inf` まで通すと意図とずれる。
  許す文字を限定し、**有限の数値だけ**を数値として扱う
- 比べる値が空のときは挙動が分かりにくい（全行一致など）ので、確認を入れる

## サンプル
- `sample/社員名簿サンプル.csv` … 部署・氏名・年齢・月給の名簿。
  例: 列を「氏名・月給」だけ残し、条件「月給 が 数値≧ 350000」で絞り込むと、高給の人だけ抽出できます。

## 注意
- 数値条件は、セルと比べる値の**両方が数値**のときだけ比較します（数値でない行は通しません）。
- 絞り込みの対象列が無い（列数が足りない）行は、結果から除外します。
