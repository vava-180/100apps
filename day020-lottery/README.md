# Day 020: おみくじ／抽選ツール

候補を入れて、ランダムに選ぶブラウザツールです。重複なしの抽選や、並べ替えもできます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 候補を1行に1つ入力する
3. ボタンを選ぶ
   - 「1つ選ぶ（おみくじ）」：くるくる回って1つに決まる
   - 「重複なしで選ぶ」：人数／個数を指定して、かぶらないように選ぶ
   - 「全部の順番を決める」：全員をシャッフルして順番を決める

## 機能
- ランダムに1つ選ぶ（回転演出つき）
- 重複なしでN個選ぶ／全体をシャッフルして順番決め
- 候補の空行・前後の空白は自動で無視

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 抽選のように「予測されては困らないが、かたよってはいけない」場面では、
  **公平なシャッフル（Fisher-Yates法）** を使うと、すべての並びが同じ確率になる
- 乱数は `crypto.getRandomValues` を使い、余り（%）で出るかたよりを取り除いている
- 演出中はボタンや入力欄を無効にして、二重クリックや候補のズレを防いでいる

## 注意
- 同じ名前を複数の行に書くと、その分だけ当たりやすくなります
  （各行を別々の「くじ」として扱うため）。
