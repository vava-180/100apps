# Day 010: 画像リサイズ・圧縮

画像を読み込んでサイズ変更・圧縮するブラウザツールです。画像は**パソコンの外に送られません**（ネット通信なし／すべてブラウザ内で処理）。

## 使い方
1. `index.html` をブラウザで開く
2. 画像をドラッグ＆ドロップ、またはクリックして選ぶ
3. 幅・高さを入力（「縦横の比率を保つ」にチェックがあれば片方を入れるだけでOK）
4. 出力形式（JPEG / PNG / WebP）と画質を選ぶ
5. 「変換する」を押すと、変換前後を並べて比較表示（どれだけ軽くなったかも表示）
6. 「ダウンロード」で保存

## 機能
- ドラッグ＆ドロップ／クリックでの画像読み込み
- 幅・高さ指定のリサイズ（縦横比の固定にも対応）
- JPEG / PNG / WebP への変換
- 画質スライダーでの圧縮（JPEG・WebP）
- 変換前後のプレビュー比較とファイルサイズの増減表示

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・**外部通信なし**）
- Canvas API（画像を描き直してリサイズ・形式変換するブラウザ標準機能）
- FileReader API（ローカルの画像を読むだけ。アップロードはしません）

## 学び・ポイント
- 画像のリサイズは「canvasに小さいサイズで描き直す → そのcanvasを画像として書き出す」で実現できる
- `canvas.toBlob(blob, 形式, 画質)` で、形式と圧縮率を指定して画像データを作れる
- 画像処理をブラウザ内だけで完結させれば、写真を外部に送らずに安全に扱える
