# Day 077: チップ計算（Flutter）

飲食代とチップ率（%）から、**チップ額・合計・1人あたり**を計算します。スライダーで割合を決め、入力のたびにリアルタイムで再計算します。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 飲食代を入力（小数OK）
- チップ率をスライダーで 0〜30% に調整
- ＋／− ボタンで人数を調整
- チップ額・合計・1人あたりをリアルタイムに表示（小数2桁＋3桁区切り）

## 使った技術
- Flutter / Dart（`Slider`・`onChanged` でのリアルタイム再計算・`toStringAsFixed(2)`）
- 追加パッケージなし

## 学び・ポイント
- 入力のたびに `setState(() {})` を呼ぶと、計算結果が即座に画面へ反映される
- 金額のような派生値は「状態として持たず」、`getter`（`_tip` など）でその都度計算するとズレない
- チップ率は割合なので `金額 × 率 ÷ 100`

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
