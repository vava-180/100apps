# Day 039: 複数Excel→1ブック集約ツール（Python）

フォルダの中にある複数のExcel（.xlsx）の各シートを、1つのExcelブックにまとめる実務ツールです。
別々のファイルに分かれた表を、1ファイルにシートとして集めたいときに使います（ローカル動作・ネット通信なし）。

## 必要なもの
- Python（3.8以上の目安）
- `openpyxl`（Excelを読み書きするライブラリ）。1回だけ次でインストール:
  ```
  python -m pip install openpyxl
  ```
  ※ `openpyxl` はネット通信をするものではなく、Excelをローカルで処理するためのライブラリです。

## 使い方
1. ターミナルで `python merge_excel.py`（WindowsならダブルクリックでもOK）
2. Excel(.xlsx)が入ったフォルダのパスを入れる
3. 集める対象が一覧表示される → `yes` で実行
4. 同じフォルダに `集約結果.xlsx` が作られる（各ファイルの各シートが入る）

## 機能
- フォルダ内の複数 .xlsx の各シートを1ブックに集約
- シート名はファイル名ベース（1ファイル1シートならファイル名、複数なら「ファイル名_シート名」）
- Excelの制限（31文字・使えない記号）と、重複（大文字小文字の違いも含む）を自動調整
- Excelの一時ファイル（`~$…`）や、過去に作った `集約結果.xlsx` は対象外
- 1ファイル／1シートが壊れていても、そこだけ飛ばして残りを集約
- 出力は連番で**上書きしない**（元のExcelは読むだけ）

## 使った技術
- Python ＋ `openpyxl`（外部通信なし）

## 学び・ポイント
- Excelのシート名には制限（31文字・`\ / ? * [ ] :` 使用不可・重複不可）がある。
  記号を `_` に、長さを31字に、重複は連番にして安全な名前を作る
- シート名の重複は**大文字小文字を区別しない**ので、比較は小文字にそろえる
- 数式は `data_only=True` で「計算結果」を読む（保存済みのファイルなら値が入る）
- 過去に作った出力を再び集めると重複・肥大化するので、**出力名のパターンを入力から除外**する
- 大きいファイルは `read_only=True` で開き、使い終わったら `close()` する

## サンプル
- `sample/` … 支店A.xlsx・支店B.xlsx・支店C.xlsx（C は2シート）。
  このフォルダを指定すると、4シート（A／B／C_第1四半期／C_第2四半期）が1つの `集約結果.xlsx` にまとまります。

## 注意
- 写すのは**セルの値だけ**です（列幅・色・罫線・結合セルなどの書式は写りません）。
- 数式は、一度Excelで開いて保存したファイルなら結果が入ります（未保存だと空のことがあります）。
