# Day 062: Outlook 一斉メール下書き作成（差し込み）（Outlook VBA）

宛先リスト（CSV）と、`{氏名}` などの差し込み欄を入れたひな形から、一人ずつの**下書きメール**をまとめて作成するマクロです。**★送信はしません★**（下書き保存のみ）。送る前に内容を必ず目視できるので安全です。

## 必要なもの
- Microsoft Outlook（デスクトップ版／VBAが使える）

## 使い方
1. 宛先CSVを用意する（1行目は見出し。メールアドレスの列が必要）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `CreateMailDrafts.bas` を取り込む
3. 件名・本文のひな形（`SUBJECT_TEMPLATE` / `BODY_TEMPLATE`）を必要に応じて書き換える
4. `Alt + F8` →「CreateMailDrafts」を実行 → CSVのパスを入力
5. 下書きフォルダに人数分の下書きができる（送信は手動）

## CSVの形式（サンプル: `sample/宛先サンプル.csv`）
```
メールアドレス,氏名,会社
taro@example.com,山田 太郎,ABC商事
hanako@example.com,佐藤 花子,DEF工業
```
- 見出しに「メール」または「mail」を含む列を、自動で宛先アドレスとして使います。
- ひな形の中で `{氏名}` `{会社}` のように **{見出し名}** と書くと、その列の値に差し替わります。

## 機能
- CSVを1行ずつ読み、行ごとに件名・本文を差し込み（プレースホルダ置換）
- メール本文・件名を入れた**下書き**を `mail.Save` で作成（送信しない）
- ダブルクォート対応の簡易CSVパーサ内蔵（`"a,b"` のようにカンマを含む値もOK）
- 空行はスキップ／アドレス列が無ければ中止

## 使った技術
- Outlook VBA（Application.CreateItem / MailItem.Save）
- ファイル入出力（Open … For Input）と自作CSVパーサ

## 学び・ポイント
- `CreateItem(0)` で新規メールを作り、`.Save` すると「下書き」に入る。`.Send` を呼ばない限り送信されないので、一斉メールは**まず下書きで作って目視**が安全
- 差し込みは「ひな形の `{見出し}` を、その行の値で `Replace`」するだけで実現できる
- CSVは単純な「カンマで分割」だと、値の中にカンマがある場合に壊れる。ダブルクォートを見ながら分割する小さなパーサにすると安全

## 注意
- 大量の下書きを一度に作ると重くなります。まずは数件で試してください。
- 文字化けを避けるため、CSVは「ANSI（cp932）」または環境に合わせた文字コードで保存してください。
- 本マクロは送信しません。送信は内容確認後、手動で行ってください。
