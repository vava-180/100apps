# Day 028: フォルダ内 横断検索ツール（Python）

指定したフォルダの中を、**ファイル名**と**中身（テキスト）**の両方からキーワードで検索する実務ツールです（外部ライブラリなし・ネット通信なし）。
中身が当たった場合は「何行目のどの行か」も表示するので、目的の箇所をすぐ探せます。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python search.py`（WindowsならダブルクリックでもOK）
2. 検索するフォルダのパスを入れる
3. 検索キーワードを入れる
4. 検索対象（名前と中身／名前だけ／中身だけ）・拡張子・サブフォルダ・大文字小文字を選ぶ
5. 結果が一覧表示される（中身ヒットは行番号付き）

## 機能
- ファイル名と中身の横断検索（対象は選べる）
- 中身ヒットは「行番号 + その行」を表示
- 文字コードは UTF-8 / Shift_JIS(cp932) を自動判別
- 画像などのバイナリ（ヌル文字を含むもの）や、大きすぎるファイル（20MB超）は中身検索の対象外
- 「.」で始まる隠しフォルダ（`.git` など）の中身は検索しない
- 権限エラーなどで読めないパスは、その分だけ飛ばして検索を続ける
- 1ファイルの表示は最大50行（多すぎる時に画面が埋まらないように）

## 使った技術
- Python（標準ライブラリ `pathlib` のみ・外部通信なし）

## 学び・ポイント
- 検索は「名前」と「中身」で別々に考えると整理しやすい
- 中身検索は1行ずつ見て、当たった行の**行番号**を一緒に出すと探しやすい
- テキストとして読めないファイル（画像など）は、ヌル文字の有無で見分けて中身検索を飛ばす
- 「自分自身（このスクリプト）」を結果から外すときは、ファイル名でなく**絶対パス**で比べると、
  同名ファイルが別フォルダにあっても誤って除外しない

## 注意
- 中身検索は、テキストとして読めるファイルが対象です（読めないものは名前だけ検索されます）。
- このツールはファイルを**書き換えません**（検索だけ）。
