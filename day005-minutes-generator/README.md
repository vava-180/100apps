# Day 005: 議事録ジェネレータ

項目を埋めるだけで、きれいな議事録テキスト（Markdown形式）を自動で組み立てるブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. 左側の項目（会議名・日付・参加者・議題・ToDoなど）を入力する
3. 右側にリアルタイムでプレビューが表示される
4. 「コピー」でMarkdownをコピーし、メールやドキュメントに貼り付ける

## 機能
- 会議名・日時・場所・参加者の整形
- 議題・決定事項を自動で番号付きリストに
- ToDoを「担当: 内容」と書くと、担当者を太字にしてチェックリスト化
- 入力に合わせてリアルタイムでプレビュー更新
- Markdownのワンクリックコピー

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 入力欄をまとめて配列で管理すると、イベント登録を1行のループで書ける
- 正規表現の「グループ取り出し（カッコ）」で「担当: 内容」を担当と内容に分けられる
