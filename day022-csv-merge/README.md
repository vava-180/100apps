# Day 022: 複数CSV結合ツール（Python）

フォルダの中にある複数のCSVを、1つのCSVにまとめる実務ツールです（外部ライブラリなし・ネット通信なし）。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python merge.py`（WindowsならダブルクリックでもOK）
2. CSVが入っているフォルダのパスを入れる
3. 「元ファイル」列を付けるか選ぶ
4. 結果が `merged.csv` として書き出される

## 機能
- フォルダ内の複数CSVを1つに結合
- 1つ目のCSVの見出しを基準にし、見出しがちがうファイルは列をそろえて結合
- 各行が「どのファイル由来か」の列を付与（任意）
- 文字コードは UTF-8 と Shift_JIS（cp932）を自動で判別

## 使った技術
- Python（標準ライブラリ `csv` のみ・外部通信なし）

## 学び・ポイント
- CSVは「見出し（先頭行）＋データ」。複数ファイルをまとめるときは
  **見出しを基準にそろえる**と、列がズレずに結合できる
- 行の列数が足りない／多いファイルがあるので、基準の列数に**そろえてから**結合する
- 文字コードは日本語環境だと UTF-8 と Shift_JIS が混在しがち。
  順に試して読み、**どちらでも読めないものは無理に読まずスキップ**して文字化けを防ぐ
- Excelで開きやすいよう、出力は UTF-8（BOM付き）にしている

## 注意
- 見出しに同じ名前の列が複数あるCSVは、どの列か区別できないためスキップします。
- 結果ファイル `merged.csv` は結合対象から除外します（大文字小文字は問いません）。
