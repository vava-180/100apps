# Day 088: 日記（Flutter）

その日のひとことを書いて保存し、**日付つき**で一覧表示します。気分（顔の絵文字）も選べます。「日時の自動付与」と「新しい順の一覧」を学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 気分（😀🙂😐😢😡）を選んで、ひとことを保存
- 保存した日時を自動で記録し、曜日つきで表示
- 一覧は新しい順／1件ずつ削除
- 本文が空のときは注意メッセージ（SnackBar）

## 使った技術
- Flutter / Dart（自作クラス DiaryEntry・`DateTime`・複数行の `TextField`）
- 追加パッケージなし

## 学び・ポイント
- 保存時に `DateTime.now()` を一緒に持たせると、日時は自動で記録できる
- `weekday` は「月=1〜日=7」。曜日リストの添字は `weekday - 1`
- 新しい順に見せたいときは、リストの**先頭に追加**（`insert(0, ...)`）すると並べ替え不要

## 注意
- データはアプリ内（メモリ）に保持するため、**アプリを終了すると消えます**。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
