# Day 033: フォルダ台帳CSV出力ツール（Python）

フォルダの中のファイルを一覧にして、「相対パス・名前・拡張子・サイズ・更新日時」を**CSV台帳**に書き出す実務ツールです（外部ライブラリなし・ネット通信なし）。
資産の棚卸しや、ファイル一覧の提出などに使えます。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python ledger.py`（WindowsならダブルクリックでもOK）
2. 一覧にしたいフォルダのパスを入れる（未入力ならこのスクリプトと同じ場所）
3. サブフォルダも対象にするか／隠しフォルダを含めるかを選ぶ
4. プレビューを確認し、`yes` で `ファイル台帳.csv` に書き出し（Excelで開きやすいUTF-8・BOM付き）

## 機能
- 1行1ファイルの台帳CSVを出力（相対パス順）
- サブフォルダの中も対象にできる（隠しフォルダは最初から降りない＝`.git` などを無駄に読まない）
- サイズは読みやすい単位（KB / MB / GB …）も併記
- 自分自身（このスクリプト）と、過去に出した台帳CSVは対象から除外
- ファイル名が `=` などで始まっても、Excelで数式扱いされないよう安全化
- 出力は連番で**上書きしない**（移動・削除もしない）

## 使った技術
- Python（標準ライブラリ `os` / `csv` / `re` / `datetime` / `pathlib` のみ・外部通信なし）

## 学び・ポイント
- `os.walk` は `dirs[:]` を書き換えると「その先へ降りない」ようにできる。
  これで隠しフォルダ除外も非再帰も、**無駄に中を読まずに**実現できる
- 「自分が出した台帳」を巻き込まないよう、除外は**自分の出力名の形だけ**に絞る
  （`ファイル台帳.csv` `ファイル台帳(2).csv` だけ。ユーザーの似た名前は残す）
- CSVに外部由来の文字（ファイル名）を書くときは、`=` `+` `-` `@` 始まりに注意。
  Excelで数式として動く事故を防ぐため、先頭に `'` を付けて無害化する

## 注意
- このツールはファイルを**移動・削除しません**（一覧をCSVに書き出すだけ）。
- 更新日時はこのPCのローカル時刻です（列名に「(ローカル)」と明記）。
