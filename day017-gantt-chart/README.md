# Day 017: ガントチャート作成

タスク名と開始日・終了日を入れると、横棒の予定表（ガントチャート）を作るブラウザツールです。データはこのブラウザに保存されます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. タスク名・開始日・終了日を入れて「追加」
3. 日付の目盛りに沿って、タスクが横棒で表示される（土日に色、今日は赤い線）
4. タスクの「×」で削除。内容はブラウザに保存され、次に開いても残る

## 機能
- タスクの追加・削除
- 開始日〜終了日を横棒（バー）で表示、日数も表示
- 土日の色分け・今日の位置を示す赤い線
- `localStorage` による自動保存（閉じても消えない）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- `localStorage`（ブラウザ内にデータを保存するしくみ）

## 学び・ポイント
- 日付の引き算は、時刻を**正午（12時）で固定**して計算すると、
  夏時間などの影響でズレるのを防げる
- 保存データ（localStorage）は**信用せずに検証する**。
  壊れたデータがあっても落ちないよう、正しい形・正しい日付のものだけ通している
- 横棒の位置は「開始日が全体の何日目か × 1日の幅」で計算して配置している

## 注意
- 表示できる期間は最大1年までです（長すぎるとマス目が増えて重くなるため）。
