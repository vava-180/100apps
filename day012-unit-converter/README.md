# Day 012: 単位換算ツール

長さ・重さ・面積・体積・速さ・温度の単位をまとめて換算するブラウザツールです（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 上のタブで種類（長さ・重さ など）を選ぶ
3. 左に数値と単位、右に変換先の単位を選ぶと、自動で換算される
4. 「⇄」で変換元と変換先を入れ替えられる

## 機能
- 6カテゴリの単位換算（長さ／重さ／面積／体積／速さ／温度）
- 変換元・変換先の入れ替え
- 温度（℃・℉・K）にも対応

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- ふつうの単位は「基準1つあたりの倍率」で表せる（例：1km＝1000m）。
  いったん基準（m など）に直してから目的の単位に変換すると、式がシンプルになる
- **温度は倍率では変換できない**（0の位置がズレるため）。
  ℃をいったん経由する変換式（toC / fromC）を単位ごとに用意して対応した
- `type="number"` の欄に「1,234」のようなカンマ付き文字列を入れると表示が壊れる。
  表示用の欄は `type="text"` にして、3桁区切りを安全に表示している

## 注意
- 面積の「畳」は約1.62㎡、「坪」は約3.305785㎡として計算しています（定義に地域差があります）。
- ケルビン(K)は0未満が存在しないため、マイナス入力はエラーになります。
