# Day 044: セル文字の一括整形（Excel VBA）

選んだ範囲（選ばなければ使用範囲）の「文字が入ったセル」だけを対象に、トリムや全角／半角の統一をまとめて行うマクロです。**数式・数値セルは触りません**。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 整えたい範囲を選ぶ（`sample/整形サンプル.xlsx` でも試せます。範囲未選択なら使用範囲が対象）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `FormatCells.bas` を取り込む
3. `Alt + F8` →「FormatCells」を実行
4. モードを選ぶ（1〜4）

## 機能（モード）
- **1 トリム**：前後の空白を消し、間の連続空白を1つに（全角スペース・タブ・ノーブレークスペースも対象）
- **2 全角→半角**：英数字・カタカナ・記号すべてを半角に（`StrConv(vbNarrow)`）
- **3 半角→全角**：英数字・カタカナ・記号すべてを全角に（`StrConv(vbWide)`）
- **4 トリム＋全角→半角**：先に全角→半角、その後に空白を整理
- 数式セル・数値セル・空セルは対象外（文字の定数セルだけ整える）
- 実行前に対象セル数を表示して確認

## 使った技術
- Excel VBA（Range.SpecialCells / StrConv / Replace）

## 学び・ポイント
- 整形したいセルは `SpecialCells(xlCellTypeConstants, xlTextValues)` で
  「文字の定数セル」だけを取り出せる（数式・数値・空セルを自動で除外）
- 全角⇔半角は `StrConv(s, vbNarrow / vbWide)` でできる。
  ただし**英数字だけでなくカナ・記号も変換される**点に注意（確認メッセージでも明示）
- `=` `+` `-` `@` で始まる文字を書き戻すと数式と解釈されることがある。
  先に書式を文字列（`"@"`）にしてから書くと、文字のまま保てる
- セルの中身は `.Value2` で読み書きすると、表示形式に左右されない

## 注意
- 元に戻せないので、必要なら先に保存してから実行してください。
- モード2・3・4は英数字以外（カナ・記号）も変換されます。住所・氏名などは結果をご確認ください。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/整形サンプル.xlsx` … 前後空白・全角英数字・全角スペースが混ざったデータ。
  モード4で「トリム＋全角→半角」を試すと、きれいにそろいます。
