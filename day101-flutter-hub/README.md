# Day 101: アプリハブ（スマホアプリ集約・非表示機能つき）

Day 66〜90 で作ったスマートフォン向けアプリ（25個）を、1つの Flutter アプリにまとめた「ハブ」です。ホームのポップなアイコンから各アプリを開けます。使わないアプリは**非表示**にでき、その設定は端末に保存されます。

## アイコン
- `assets/icon.png` … 3×3のカラフルなタイルを並べた、ポップなアプリアイコン（自作）。アプリ内のイメージにも使えます。

## 必要なもの
- Flutter（デスクトップ/モバイルのいずれか）

## 使い方（開発者向け）
1. このフォルダで `flutter pub get`
2. `flutter run`（実機・エミュレータ・対応デスクトップ）
3. ホームのアイコンをタップして各アプリを使う

## 機能
- **25アプリを集約**：カウンター / メモ / ToDo / 買い物 / 家計簿 / 体重 / 習慣 / ストップウォッチ / ポモドーロ / BMI / 割り勘 / チップ / 単位変換 / 為替 / パスワード / QR / カラー / おみくじ / サイコロ / ガチャ / 名言 / 単語帳 / 日記 / あいさつ / 電卓
- **非表示機能**：右上の「編集」でカードの×から非表示に。右上の管理画面でいつでも表示/非表示を切り替え
- **設定の保存**：非表示の状態は `shared_preferences` で端末に保存（次回起動でも維持）
- **全部表示**ボタンで、隠したアプリをまとめて戻せる

## 使った技術
- Flutter / Dart（Material 3）
- `shared_preferences`（非表示設定の保存・オフライン）
- `qr_flutter`（QRコード描画・オフライン）

## 学び・ポイント
- 各アプリを `MiniApp`（番号・名前・絵文字・色・画面）のリストにすると、ホームのグリッドや非表示管理をデータ起点で書ける
- 非表示の状態は `Set<int>` で持ち、`shared_preferences` に文字列リストとして保存・復元する
- `Timer` は `dispose` で必ず `cancel` する（ストップウォッチ・ポモドーロ）
- 非同期処理（保存の読み込み）後に `setState` する前は `mounted` を確認する
- パスワードは `Random.secure()`、電卓は0除算をチェックして「エラー」を表示

## 注意
- ネット通信はありません（追加パッケージもすべてオフライン動作）。
- 各アプリのデータ（メモ・ToDoなど）はこのデモではメモリ上のみで、アプリを閉じると消えます（非表示設定だけ保存されます）。
