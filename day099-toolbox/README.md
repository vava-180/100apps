# Day 099: まとめツール（便利ツール集）

これまで作った人気のブラウザツールを、タブで切り替えて使える「便利ツール集」に合体させました。1ファイル・オフラインで動きます。

## 使い方
1. `index.html` をブラウザで開く
2. 上のタブで使いたいツールを選ぶ
3. 各ツールを操作する

## 入っているツール
- **文字数カウント**：文字数・空白を除いた文字数・行数をリアルタイム表示（絵文字も1文字）
- **割り勘**：合計金額と人数から、1人あたり（切り上げ）と差額を計算
- **パスワード生成**：長さと文字種を選んで、安全な乱数で生成
- **単位変換（長さ）**：mm / cm / m / km / inch / feet を相互変換
- **カラーコード**：RGBスライダーから HEX を表示＆色プレビュー

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- タブUIは「ボタンのdata属性」と「パネルのid」を対応させ、`active` クラスを付け替えるだけで作れる
- パスワードは `Math.random()` ではなく `crypto.getRandomValues()` を使うと、より安全な乱数になる
- 割り勘は `Math.ceil()` で切り上げ、`1人あたり × 人数 − 合計` でおつり（差額）が分かる
- 単位変換は「いったんメートルに直す → 目的の単位に直す」と、どの組み合わせでも計算が1本化できる
- 入力チェックは `Number.isFinite`（数値か）と `Number.isInteger`（整数か）を使い分ける

## 注意
- 完全にオフラインで動きます（外部APIやネット通信はありません）。
