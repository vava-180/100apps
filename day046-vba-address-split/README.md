# Day 046: 住所分割（都道府県／市区町村以降）（Excel VBA）

選んだ1列の住所を、「都道府県」と「それ以降（市区町村以降）」に分けて、すぐ右の2つの列に書き出すマクロです。

例: A列「東京都新宿区西新宿2-8-1」→ B列「東京都」／C列「新宿区西新宿2-8-1」

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 住所が入った「1列分」を選ぶ（`sample/住所サンプル.xlsx` でも試せます。見出しは選択から外すのがおすすめ）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `SplitAddress.bas` を取り込む
3. `Alt + F8` →「SplitAddressMain」を実行
4. 右の2列に「都道府県」「市区町村以降」が入る

## 機能
- 47都道府県リストとの**前方一致**で都道府県を判定（4文字県も正しく分割）
- 都道府県が無い住所は、そのまま「市区町村以降」に入れる
- 全角スペースを半角にしてからトリム（先頭の全角空白で判定が崩れない）
- 右2列に既存値（数式・エラー値含む）があれば、上書き前に確認
- 1列・ひと続きの選択に限定（右に2列の空きが必要）
- 空セル・エラー値セルは飛ばす

## 使った技術
- Excel VBA（前方一致 / Array / Range.Offset / Intersect）

## 学び・ポイント
- 都道府県の判定は「`県`で終わるか」などの文字ルールより、**47都道府県のリストと前方一致**が確実。
  神奈川県・和歌山県・鹿児島県のような4文字県も、リストなら正しく分かれる
- どの都道府県も他の都道府県の接頭辞になっていないので、前方一致で取り違えは起きない
- 都道府県名リストは**一度だけ作って使い回す**と、行数が多くても無駄がない
- 上書き判定は「表示が空でも数式が入っている」セルも見落とさないようにする

## 注意
- 分けるのは「都道府県」までです。市区町村と番地のさらに細かい分割はしません。
- 旧字体・異体字（例「鹿兒島縣」）には対応していません（標準表記が前提）。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/住所サンプル.xlsx` … 各種の住所（4文字県・北海道・府なしを含む）。
  A列を選んで実行すると、B・C列に分割結果が入ります。
