# Day 094: Markdown → HTML 変換

左に Markdown を書くと、右に HTML プレビューがリアルタイムで出るブラウザツールです。外部ライブラリを使わず、自前の小さなパーサで変換します。

## 使い方
1. `index.html` をブラウザで開く
2. 左のエリアに Markdown を書く（最初に見本が入っています）
3. 右にプレビューがすぐ表示される
4. 「HTMLコードを表示」で変換結果のHTMLを確認・コピーできる

## 機能
- リアルタイムプレビュー
- 対応記法：見出し（#）/ 太字（\*\*）/ 斜体（\*）/ コード（\`）/ リンク / 箇条書き / 番号付きリスト / 引用（>）/ 水平線 / コードブロック（\`\`\`）
- 生成したHTMLの表示＆クリップボードコピー
- XSS対策：先にHTMLエスケープし、リンクは安全なURL形式だけ `<a>` 化

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- `innerHTML` に入れる前に `escapeHtml()` で `< > & "` を実体参照に変えると、タグ混入（XSS）を防げる
- リンクURLは `http(s)/mailto/相対` だけを許可（allowlist）して `javascript:` などを弾く
- コード部分は先に切り分けてから装飾すると、`` `a*b*c` `` のアスタリスクを誤って太字にしない
- 太字は最短一致 `\*\*(.+?)\*\*`、斜体は `\*([^*]+)\*` の順にすると、太字の中の斜体も処理できる

## 注意
- 完全にオフラインで動きます（外部APIやネット通信はありません）。
- 簡易Markdownです。リストの継続行・ネストリスト・URL内の括弧などは未対応です。
