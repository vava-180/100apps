# Day 097: 100apps 記録サイト（一覧自動生成）

プロジェクト直下の「dayXXX-アプリ名」フォルダを全部さがして、各 README からタイトルを読み取り、フェーズ別にならべた一覧ページ（`index.html`）を**自動生成**するスクリプトです。

## 必要なもの
- Python 3（標準ライブラリのみ・追加インストール不要）

## 使い方
1. ターミナルでこのフォルダに移動する
2. 実行する
   - `python generate_site.py`（プロジェクト直下を対象に生成）
   - `python generate_site.py 対象フォルダ`（対象を指定）
3. このフォルダに `index.html` ができる → ブラウザで開くと全アプリの一覧が見られる

## 機能
- `dayXXX-...` フォルダを自動で収集し、Day番号順にならべる
- 各 README.md の大見出し（H1）をタイトルとして表示
- フェーズ（1〜5）ごとに見出しを付けてカード表示
- 各カードはアプリ本体（index.html）または README.md へのリンク付き
- `note_object` 系フォルダは除外（ルール厳守）

## 使った技術
- Python（pathlib / re / html / urllib.parse）

## 学び・ポイント
- 画面に出す文字は `html.escape()`、リンクのURLは `urllib.parse.quote()` と、**用途で使い分ける**と二重エスケープを防げる
- READMEは `utf-8-sig` で読むとBOM付きにも対応でき、`UnicodeError` も `except` で受けて全体が止まらないようにする
- タイトルは「H1だけ」を正規表現 `^#\s+(.+)$` で取ると、`##` の小見出しを誤って拾わない
- ソートキーを `(番号, フォルダ名)` にすると、同じ番号でも順番が安定する

## 注意
- 完全にローカルで動きます（外部APIやネット通信はありません）。
- 生成される `index.html` は、このフォルダから見て1つ上の階層の各アプリにリンクします。
