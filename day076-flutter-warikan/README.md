# Day 076: 割り勘（Flutter）

合計金額と人数から「**1人あたり**」を計算します。端数は **100円単位で切り上げ**て払いやすくし、集まる合計とおつりも表示します。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 合計金額を入力し、＋／− ボタンで人数を調整
- 1人あたりを「100円単位で切り上げ」て表示
- 集まる合計・おつり（集まり − 合計）も表示
- 金額が未入力・0以下のときは注意メッセージ（SnackBar）

## 使った技術
- Flutter / Dart（`int.tryParse`・`ceil()` による切り上げ・3桁区切り自作）
- 追加パッケージなし

## 学び・ポイント
- 「払いやすさ」を考えると、ピッタリ割るより **100円単位に切り上げ**ると現場で使いやすい（`(値 ÷ 100).ceil() × 100`）
- 切り上げた分は集まりすぎるので、「おつり」を出すと納得感がある
- 人数は 1 人未満にならないよう、− ボタンを `_people > 1` で無効化

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
