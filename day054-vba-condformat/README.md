# Day 054: 条件付き書式の一括設定（Excel VBA）

選んだ範囲（選ばなければ使用範囲）に、条件付き書式をまとめて設定するマクロです。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 設定したい範囲を選ぶ（`sample/条件付き書式サンプル.xlsx` の点数列で試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `ApplyConditionalFormat.bas` を取り込む
3. `Alt + F8` →「ApplyConditionalFormat」を実行
4. 種類（1〜4）を選ぶ

## 機能（種類）
- **1**：しきい値より大きいセルを黄色に
- **2**：しきい値より小さいセルを黄色に
- **3**：重複している値を赤系に
- **4**：カラースケール（小さい＝赤 〜 中＝黄 〜 大きい＝緑）
- 設定前に、その範囲の既存の条件付き書式を消す（重ね掛け防止）
- 1セルだけ／未選択なら使用範囲を対象。バラバラ（非連続）選択は断る

## 使った技術
- Excel VBA（Range.FormatConditions の Add / AddUniqueValues / AddColorScale）

## 学び・ポイント
- しきい値で色を付けるのは `FormatConditions.Add(xlCellValue, xlGreater, 値)`。
  値は**数値のまま渡す**と、小数点の記号がロケール（地域設定）で崩れない
- 重複は `AddUniqueValues` ＋ `DupeUnique = xlDuplicate` で色付けできる
- カラースケールは `AddColorScale(3)` で3色。最小＝赤・中央(50%)＝黄・最大＝緑、のように指定する
- 条件付き書式は重ねて設定されやすいので、先に `FormatConditions.Delete` で消すと分かりやすい

## 注意
- 既存の条件付き書式は、対象範囲ぶんだけ消してから設定します。
- バラバラ選択（Ctrlで複数箇所）には対応していません（ひと続きの範囲を選んでください）。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/条件付き書式サンプル.xlsx` … 7人の点数。B列を選んで、
  「しきい値60より大きい」や「カラースケール」を試せます。
