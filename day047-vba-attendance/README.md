# Day 047: 勤怠集計マクロ（Excel VBA）

勤怠表から、各行の「実働時間」を計算して列に書き出し、合計を下に追記するマクロです。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 表の形（見出し）
- 必要：「出勤」「退勤」（Excelの時刻として入力。例 `9:00`）
- 任意：「休憩」（分。数値。例 `60`）
- 出力：「実働」（無ければ右端に「実働(時間)」を自動作成）

## 使い方
1. 勤怠表のあるシートを開く（`sample/勤怠サンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `SummarizeAttendance.bas` を取り込む
3. `Alt + F8` →「SummarizeAttendance」を実行

## 機能
- 各行の実働時間 ＝（退勤 − 出勤）− 休憩 を計算（時間の小数で出力。例 7.5）
- 退勤が出勤より早い行は「日をまたいだ勤務」として計算
- 合計を表の下に追記
- 見出しは**完全一致**で検出（「実働メモ」などを誤検出しない）
- 実働列の自動作成は「実行する」を押した後に行う
- 時刻が読めない行は飛ばして古い値を消す／休憩が実働より長い行は0にして件数表示
- マイナスの休憩は0扱い／保護シートは事前に停止

## 使った技術
- Excel VBA（Range.Find / 時刻シリアル値の計算 / WorksheetFunction.Round）

## 学び・ポイント
- Excelの時刻は「1日＝1.0」の割合。`退勤 − 出勤` を `×24` すると時間になる
- 日をまたぐ勤務は、差がマイナスになるので **+1日（+1.0）** して直す
- 見出しの列探しは、`xlPart`（部分一致）だと「実働メモ」まで拾うので、**完全一致＋候補リスト**が安全
- 合計行のラベルを「出勤」列に書くと、次回実行の最終行判定に巻き込まれる。**汚さない場所**に置く
- セルに書く値と合計に足す値は、**同じ丸め済みの値**にすると表示と合計がズレない

## 注意
- 「休憩」は分（数値）で入力してください（`1:00` のような時刻形式ではありません）。
- 「出勤」「退勤」はExcelの時刻として入力してください（`9:00` など）。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/勤怠サンプル.xlsx` … 5日分の勤怠（日跨ぎ勤務を含む）。
  実行すると実働(時間)列が埋まり、下に合計（33.5時間）が出ます。
