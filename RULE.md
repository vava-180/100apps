# RULE.md（厳守ルール）

Claudeはこのプロジェクトで以下を必ず守る。

## 1. 外部APIは許可制（最重要）
- **IMPORTANT: 外部API（ネットの向こうのサービスを呼ぶ処理）を勝手に使わない。**
- 必要な場合は、実装前に必ず理由を説明し、許可を得てから使う。
- 補足: アプリに同梱して動くJSライブラリ（例: QRコード生成など、ネット通信しないもの）はAPIに当たらず使ってよい。判断に迷うものは確認する。

## 2. 小さく作る
- 1つのアプリは1日（目安30分）で完成する規模に保つ。
- 頼まれていない機能を勝手に足さない。大きなリファクタも勝手にしない。

## 3. 安全
- **IMPORTANT: パスワード・トークン・APIキーなどの秘密情報を、コードやコミットに含めない。**

## 4. 初心者サポート
- 変更の前に「何をするか」を一言説明する。
- 専門用語は初心者向けにかみ砕いて説明する。

## 5.触らないフォルダ
- 以下フォルダは無視すること
