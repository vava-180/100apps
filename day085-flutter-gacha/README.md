# Day 085: 抽選・ガチャ（Flutter）

ボタンを引くと、レア度ごとの「**当たりやすさ（重み）**」に従って結果が出るガチャです。出やすさに差をつける「重み付き抽選」を学べます。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- ガチャを引くと、レア度（★5〜★1）が重みに従って当たる
- ★5=1%／★4=5%／★3=14%／★2=30%／★1=50% の確率（合計100%）
- 引いた回数をカウント／確率も画面に表示

## 使った技術
- Flutter / Dart（`Random`・`fold` による重み合計・重み付き抽選ロジック）
- 追加パッケージなし

## 学び・ポイント
- 「重み付き抽選」は、重みの合計の中から1点をランダムに選び、**どの景品の範囲に入るか**を順に調べる方法で実現できる
- 重みを「%」にそろえておくと（合計100）、そのまま確率として説明しやすい
- 単純な等確率の抽選（おみくじ）との違いがポイント

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
