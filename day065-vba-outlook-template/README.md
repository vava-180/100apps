# Day 065: 定型メールのテンプレ送信補助（Outlook VBA）

よく使う定型文（お礼／日程調整／資料送付）から1つ選ぶと、件名・本文を入れた新規メールを**画面に表示**するマクロです。**★自動送信はしません★**。宛名を差し込み、内容を確認・微修正してから自分で送信できるので安全です。

## 必要なもの
- Microsoft Outlook（デスクトップ版／VBAが使える）

## 使い方
1. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `TemplateMail.bas` を取り込む
2. `Alt + F8` →「TemplateMail」を実行
3. テンプレ番号（1〜3）と宛名を入力 → 下書きが画面に開く
4. 内容を確認・修正して、自分で送信ボタンを押す

## 同梱テンプレート
1. お礼メール
2. 日程調整のお願い
3. 資料送付のご案内

- 各テンプレの `{宛名}` は、実行時に入力した名前へ差し替わります（空欄なら「ご担当者」）。
- テンプレは `GetTemplate` 内の `Select Case` に追記すれば、いくらでも増やせます。

## 使った技術
- Outlook VBA（Application.CreateItem / MailItem.Display）

## 学び・ポイント
- `.Display` はメールを「表示」するだけで送信しない。定型文を入れた状態で開けるので、確認してから送れる（`.Send` を呼ばない限り送信されない）
- テンプレ本体（件名・本文）を1つの関数にまとめ、`ByRef` で件名・本文を返すと、呼び出し側がすっきりする。テンプレ追加も1か所で済む
- `{宛名}` のような差し込み欄を1つ用意するだけでも、定型メールはぐっと使いやすくなる
- 署名を自動挿入する設定にしていれば、`.Display` で開いたメールにも署名が付く

## 注意
- 本マクロは送信しません。内容確認後、手動で送信してください。
- 宛先（To）は空のまま開きます。送信前に必ず宛先を入れてください。
