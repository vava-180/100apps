# Day 091: 数当てゲーム

コンピューターが決めた数字を、ヒントを頼りに当てるブラウザゲームです。フェーズ5（自由選択）の1本目として、すぐ遊べてネット通信不要の構成にしました。

## 使い方
1. `index.html` をブラウザで開く
2. むずかしさ（1〜100 / 1〜500 / 1〜1000）を選ぶ
3. 数字を入力して「答える」（Enterキーでも可）
4. 「もっと大きい／小さい」のヒントを見ながら、正解を目指す

## 機能
- 1〜選んだ上限 のランダムな整数を当てるゲーム
- 「もっと大きい／小さい／正解」をヒント表示（色分け）
- 挑戦回数のカウント、試した数字の履歴表示
- むずかしさ（範囲）を3段階から選択
- 入力チェック（空欄・小数・文字・範囲外・同じ数字の二度入力を弾く）
- 「新しいゲーム」で答えを決め直してやり直し

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- ランダムな整数は `Math.floor(Math.random() * maxNum) + 1` で「1〜maxNum」を作れる
  （`Math.random()` は 0以上1未満なので、`+1` で下限を1に合わせる）
- 入力チェックは `Number.isInteger()` で「整数かどうか」を確かめると、小数や文字をきちんと弾ける
- 同じ数字の二度入力を記録（配列）で防ぐと、回数が不正に増えない
- 正解後は入力欄とボタンを `disabled` にして、操作を止めると分かりやすい

## 注意
- 完全にオフラインで動きます（外部APIやネット通信はありません）。
