# Day 072: 習慣トラッカー（Flutter）

続けたい習慣を登録し、「今日やった」を押すと**通算の達成回数**が増えていくアプリです。積み上がった回数を見て、継続のモチベーションにします。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 習慣を追加／削除
- タップで「今日やった」を切り替え（達成すると通算回数 +1、取り消すと -1）
- 今日の達成数（◯ / 全件）を表示
- ヘッダーの更新ボタンで「今日の達成」だけリセット（通算回数は残す）

## 使った技術
- Flutter / Dart（自作クラス Habit・where によるカウント）
- 追加パッケージなし

## 学び・ポイント
- 「今日やったか（doneToday）」と「通算回数（total）」を分けて持つと、翌日リセットしても積み上げが消えない
- リストの件数集計は `where(...).length` が簡単
- 同じ操作を「ボタン」と「行タップ」の両方に割り当てると使いやすい

## 注意
- データはアプリ内（メモリ）に保持するため、**アプリを終了すると消えます**。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
