# Day 006: 表→Markdown / HTML変換

Excelやスプレッドシートからコピーした表を貼り付けると、Markdown表やHTML表に変換するブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. ExcelやGoogleスプレッドシートで表をコピーして貼り付ける（タブ区切り）
3. 「Markdown」「HTML」タブで出力形式を切り替える
4. 「コピー」で結果をコピーできる（下にプレビュー表も表示）

## 機能
- タブ区切り／カンマ区切りの自動判定
- Markdown表への変換
- HTML表への変換（thead / tbody 付き）
- 1行目を見出しにするかの切り替え
- 変換結果のプレビュー表示
- `|` や `< >` などの記号を安全にエスケープ

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- Excelのコピーは「タブ区切りテキスト」になるので、`split('\t')` で列に分けられる
- HTMLに文字を出すときは `< > &` をエスケープしないと表示が崩れる（セキュリティの基本でもある）
