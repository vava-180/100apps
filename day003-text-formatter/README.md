# Day 003: テキスト整形ツール

コピペした文章の「いらない空白・空行・改行」をワンクリックで掃除するブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. 整形したいテキストを貼り付ける
3. 使いたい整形メニュー（複数選べる）を押して選ぶ
4. 「整形して下に表示」を押す
5. 「コピー」で結果をコピーできる

## 機能（整形メニュー）
- 行頭・行末の空白を削除
- 連続スペースを1つにまとめる
- 連続する空行を1つにまとめる
- 空行をすべて削除
- 改行をなくして1行につなげる
- 全角スペースを半角スペースに変換
- メニューは複数選んでまとめて適用できる

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 整形処理を「名前：関数」のまとまり（オブジェクト）にしておくと、ボタンと処理をきれいに結びつけられる
- 正規表現の `{2,}` は「2回以上の繰り返し」。連続スペースや空行をまとめるのに便利
