# Day 092: タイピング練習

表示された英文をタイプして、CPM（1分あたり文字数）・正確率・タイムを測るブラウザ練習ツールです。正しいキーを押さないと次の文字へ進まない方式なので、IME（日本語入力）なしで気軽に測れます。

## 使い方
1. `index.html` をブラウザで開く
2. 「スタート」を押す（IME＝日本語入力はオフにしておく）
3. 表示エリアに出る英文を、上から順にタイプする
4. 全5問を打ち終えると、タイムと正確率が表示される

## 機能
- 1文字ずつ色分け表示（打ち終え＝緑／今打つ文字＝カーソル／間違い＝赤）
- CPM・正確率・経過秒・問題番号をリアルタイム表示
- 出題はスタートのたびにシャッフル
- 正しいキーでだけ前進、間違いは正確率に反映
- Backspace でやり直し可能（取り消した分は正解数も戻すので数値が水増しされない）
- 日本語入力(IME)・キー長押しは数えない（正確率が壊れないように）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- `keydown` の `e.key` が1文字（`length === 1`）のときだけ「文字キー」として扱える
- `e.isComposing` や `e.repeat` を見て、IME変換中・長押しを除外すると測定が安定する
- 「未開始」は `0` ではなく `null` で表すと、`if (!startTime)` の落とし穴（0をfalsyと誤判定）を避けられる
- Backspace で進み具合を戻すなら、正解カウントも一緒に戻さないと CPM・正確率がズレる

## 注意
- 完全にオフラインで動きます（外部APIやネット通信はありません）。
- 日本語入力（IME）はオフにしてお使いください。
