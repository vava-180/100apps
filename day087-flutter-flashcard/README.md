# Day 087: 単語帳（フラッシュカード）（Flutter）

「表（問題）」と「裏（答え）」のカードを登録し、**タップでめくって**覚える単語帳です。カードの「現在位置」と「表裏」を状態として持つ練習になります。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- カードをタップして表（問題）⇄裏（答え）をめくる
- 「前へ／次へ」でカードを移動（移動すると表に戻る）
- ＋ボタンで新しいカード（表・裏）を追加
- 何枚目かを表示（◯ / 全枚数）

## 使った技術
- Flutter / Dart（自作クラス Card2・`GestureDetector`・`showDialog`・`clamp`）
- 追加パッケージなし

## 学び・ポイント
- 「今どのカードか（index）」と「表か裏か（showBack）」の2つを状態として持つと、めくり・移動が素直に書ける
- 番号が範囲外にならないよう `clamp(0, 枚数-1)` で安全に丸める
- 入力ダイアログは `showDialog` ＋ `AlertDialog`。閉じるのは `Navigator.pop`

## 注意
- データはアプリ内（メモリ）に保持するため、**アプリを終了すると消えます**。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
