# Day 024: フォルダ自動仕分けツール（Python）

散らかったフォルダの中のファイルを、サブフォルダに自動で仕分ける実務ツールです（外部ライブラリなし・ネット通信なし）。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python organize.py`（WindowsならダブルクリックでもOK）
2. 仕分けしたいフォルダのパスを入れる
3. 仕分け方法を選ぶ
   - 1) 種類別（画像・文書・動画・音声・圧縮・その他）
   - 2) 拡張子別（JPG / PDF / XLSX …）
   - 3) 日付別（更新日の「年-月」）
4. プレビューを確認し、`yes` で実行

## 機能
- 3つの仕分け方法（種類別／拡張子別／日付別）
- 実行前のプレビュー（どのフォルダに何件入るか）と確認
- 同名ファイルは上書きせず、自動で連番を付与
- 途中で失敗したら、どこへ移動済みかを一覧表示

## 使った技術
- Python（標準ライブラリ `pathlib` / `datetime` のみ・外部通信なし）

## 学び・ポイント
- 「拡張子 → 種類」の対応表を作っておくと、ファイルを種類ごとに振り分けられる
- 移動の前に**プレビューと確認**を入れると、思っていたのとちがう仕分けを防げる
- 同名ファイルがあると上書きされて消える危険があるので、**連番を付けて回避**する
- 日付別はこのパソコンの時刻設定（ローカルタイム）の更新日で分類する

## 注意
- 対象はフォルダ直下のファイルのみ（サブフォルダ・「.」で始まる隠しファイルは対象外）。
