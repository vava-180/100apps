# Day 013: カラーピッカー＆パレット

色を選ぶと、HEX / RGB / HSL のカラーコードと、配色候補を表示するブラウザツールです（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 色をクリックして選ぶ（または HEX を直接入力）
3. HEX / RGB / HSL のコードが表示される。「コピー」で各コードをコピー
4. 下に出る配色候補（明るさ違い・調和色）をクリックすると、その色に切り替わる

## 機能
- 色の選択（カラーピッカー／HEX手入力）
- HEX・RGB・HSL の相互変換と表示・コピー
- 配色候補の自動生成（明るさ違いのグラデーション／色相をずらした調和色）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 色は HEX・RGB・HSL という3つの表し方があり、計算で行き来できる
- 配色は **HSL** で考えると作りやすい。
  明るさ(L)だけ変える＝グラデーション、色相(H)をずらす＝調和色
- 表示のために数値を丸めると、その丸めた値で配色を作ったとき元の色とズレてしまう。
  **計算用は丸めず、表示するときだけ丸める**のがコツ（往復しても色が変わらない）
