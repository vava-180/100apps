# Day 079: 為替計算（Flutter）

レート（1ドル＝何円か）を**手で入力**して、金額を「円 ⇄ ドル」で変換します。**ネット通信はしません**（外部APIは使わず、レートは自分で入力する方式）。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- レート（1ドル＝◯円）を手入力（初期値150）
- 「円 → ドル」「ドル → 円」を切り替え
- 金額を入力するとリアルタイムで変換（小数2桁＋3桁区切り）

## 使った技術
- Flutter / Dart（`SegmentedButton`・`double.tryParse`・getter で都度計算）
- 追加パッケージなし・**ネット通信なし**

## 学び・ポイント
- 円→ドルは「円 ÷ レート」、ドル→円は「ドル × レート」。方向で割り算・掛け算が逆になる
- レートを最新にしたいときは外部APIも使えるが、本チャレンジのルールに沿って**手入力**にした（通信なしで完結）
- レートが 0 以下や金額が空のときは結果を出さない（`null` で表現）

## 注意
- レートは自動更新されません（自分で入力してください）。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
