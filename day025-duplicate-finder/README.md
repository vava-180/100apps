# Day 025: 重複ファイル検出ツール（Python）

フォルダの中から「中身がまったく同じファイル」を見つける実務ツールです。
名前がちがっても、中身が同じならコピー（重複）として検出します（外部ライブラリなし・ネット通信なし）。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python find_dup.py`（WindowsならダブルクリックでもOK）
2. 調べたいフォルダのパスを入れる
3. サブフォルダも調べるか選ぶ
4. 重複の一覧が表示される
5. 希望すれば「1つだけ残して、残りを `_duplicates` フォルダへ移動」（削除はしません）

## 機能
- 中身が同じファイルの検出（サイズ → SHA-256ハッシュの2段構え）
- 重複グループの一覧表示（残す1つ／移動候補が分かる）
- 重複ぶんを `_duplicates` へ移動（削除ではなく移動）
- 移動した元の場所を `duplicates_manifest.csv` に記録（あとで戻せる）

## 使った技術
- Python（標準ライブラリ `hashlib` / `csv` のみ・外部通信なし）

## 学び・ポイント
- 「中身が同じ」は、ファイルの内容から計算する**ハッシュ（SHA-256）**で判定できる
- 全部のファイルをハッシュ計算すると遅いので、**まずサイズで仲間分け**して、
  同じサイズのものだけハッシュを計算すると速い（サイズがちがえば中身もちがう）
- いきなり削除せず**移動**にし、さらに**元の場所を記録**しておくと、間違えても戻せる
- 読み取れないファイルが1つあっても全体を止めず、そのファイルだけ飛ばす

## 注意
- 移動は削除ではありません。`_duplicates` フォルダを確認してから消してください。
- 再実行しても、前回の `_duplicates` の中は調べ直しません（二重移動を防止）。
