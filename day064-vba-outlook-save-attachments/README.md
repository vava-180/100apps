# Day 064: 添付ファイル一括保存（Outlook VBA）

選択したメール（複数選択OK）の**添付ファイル**を、指定フォルダへまとめて保存するマクロです。同名ファイルがぶつからないよう、必要なら自動で連番を付けます。請求書や写真など、添付の取りまとめに便利です。

## 必要なもの
- Microsoft Outlook（デスクトップ版／VBAが使える）

## 使い方
1. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `SaveAttachments.bas` を取り込む
2. コード内の `SAVE_FOLDER`（保存先フォルダ）を自分の環境に書き換える
3. Outlookの一覧で、保存したいメールを選ぶ（`Ctrl` で複数選択可）
4. `Alt + F8` →「SaveAttachments」を実行

## 機能
- 選択中の複数メールから、添付ファイルをまとめて保存
- 保存先フォルダが無ければ、途中の階層ごと自動作成
- 同名ファイルは `名前(2).拡張子` のように連番を付けて上書きを防止
- 署名の小さな画像などを拾わないよう、サイズ下限（`MIN_SIZE`、初期8KB）で除外（0で全部保存）

## 使った技術
- Outlook VBA（ActiveExplorer.Selection / Attachments.SaveAsFile）
- Scripting.FileSystemObject（フォルダ作成・パス操作）

## 学び・ポイント
- 「いま選択しているメール」は `Application.ActiveExplorer.Selection` で取れる。一覧で複数選んでから実行すれば一括処理になる
- 添付は `mail.Attachments.Item(j).SaveAsFile パス` で保存できる。メール署名に埋め込まれた小さな画像も「添付」扱いなので、サイズで除外すると実用的
- 同名ファイルの上書き事故を防ぐには、保存前に存在チェックして連番を付ける小さな関数を用意するとよい
- フォルダ作成は、親フォルダから順に再帰的に作ると、途中の階層が無くても失敗しない

## 注意
- メール本体は変更しません（添付を保存するだけ）。
- 信頼できる送信元の添付だけを保存してください（添付ファイルは中身を確認するまで開かない）。
- `MIN_SIZE` を大きくしすぎると、必要な小さい添付も除外されます。
