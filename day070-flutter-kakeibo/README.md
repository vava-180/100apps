# Day 070: 家計簿（Flutter）

「収入」「支出」を内容と金額で記録し、**合計（残高）を上に表示**する簡単家計簿です。入力した金額の合計を計算して見やすく整える＝**数値の集計とフォーマット**の練習になります。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 「支出／収入」を切り替えて、内容＋金額を記録
- 残高（収入合計 − 支出合計）をカードで大きく表示（マイナスは赤）
- 金額は3桁区切りで表示（`¥1,234`）／1件ずつ削除
- 金額が未入力・0以下のときは注意メッセージ（SnackBar）

## 使った技術
- Flutter / Dart（自作クラス Entry・SegmentedButton・int.tryParse・SnackBar）
- 追加パッケージなし（3桁区切りも自作）

## 学び・ポイント
- 入力された金額は文字列なので、`int.tryParse` で数値に変換し、失敗（null）や0以下をはじくと安全
- 「支出/収入」のような二択は `SegmentedButton` が分かりやすい
- 3桁区切りは `intl` パッケージでもできるが、ここでは**追加パッケージ無し**で実現（後ろから3桁ごとにカンマを入れる）
- 合計は「収入は＋、支出は−」で1つの残高にまとめると、状態がシンプルになる

## 注意
- データはアプリ内（メモリ）に保持するため、**アプリを終了すると消えます**（保存は今後の拡張ポイント）。
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
