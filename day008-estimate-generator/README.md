# Day 008: 簡易見積書ジェネレータ

品目と金額を入れるだけで、小計・消費税・合計を自動計算する見積書ツールです。そのまま印刷してPDF保存もできます。

## 使い方
1. `index.html` をブラウザで開く
2. 宛先・件名・発行日・自社名を入力する
3. 品目・単価・数量を入力する（「＋ 行を追加」で行を増やせる）
4. 税率を設定すると、小計・消費税・合計が自動計算される
5. 「🖨 印刷 / PDF保存」で見積書として出力（ブラウザの「PDFに保存」でPDF化）

## 機能
- 品目行の追加・削除
- 単価 × 数量で金額を自動計算
- 小計・消費税（税率変更可）・合計の自動計算
- 金額を「¥1,234」の3桁区切りで表示
- 印刷用レイアウト（入力欄の枠を消して紙の見積書らしく出力）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 行を動的に追加するときは、その行の入力欄に対してまとめてイベントを登録すると管理が楽
- `toLocaleString('ja-JP')` で数字を「1,234」の形にできる
- `@media print` で画面用と印刷用の見た目を分けられる
