# Day 019: ポモドーロタイマー

25分の作業と5分の休憩を自動でくり返す集中タイマーです。終了時に音で知らせます（ネット通信なし）。

## 使い方
1. `index.html` をブラウザで開く
2. 「スタート」で作業の25分が始まる（円が減っていく）
3. 0になると音が鳴り、自動で休憩に切り替わる。休憩が終わると次の作業へ
4. 「一時停止／再開」「リセット」で操作。作業・休憩の長さは下の設定で変更可

## 機能
- 作業／休憩の自動くり返し（フェーズで色が変わる）
- 円グラフで残り時間を表示、完了した作業回数のカウント
- 作業・休憩の長さを設定（分）
- 終了音（Web Audioでブラウザ内生成）

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）
- SVGの円で進捗リングを表現、Web Audio APIで終了音

## 学び・ポイント
- カウントダウンは終了予定の時刻（endAt）を基準にし、
  画面を閉じ忘れてスリープしても、**過ぎたフェーズをまとめて消化**してズレを防ぐ
  （`endAt += フェーズの長さ` を、追いつくまでくり返す）
- 動作中は設定欄を触れないようにし、途中で時間が変わって混乱するのを防いでいる
- 設定値は範囲内の整数に直して入力欄にも書き戻し、表示と実際の時間を一致させる
