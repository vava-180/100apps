# Day 086: 名言表示（Flutter）

ボタンを押すたびに、用意した名言をランダムに表示します。「**直前と同じものを出さない**」ちょっとした工夫がポイントです。

## 必要なもの
- Flutter SDK／実機・エミュレータ・Windowsデスクトップのいずれか

## 実行手順
このフォルダには `lib/main.dart` と `pubspec.yaml` だけを置いています。

```bash
flutter create .   # 足場を生成
flutter run        # 実行
```
> `flutter create .` が `lib/main.dart`・`pubspec.yaml` を上書きしたら、本リポジトリの内容に戻してください。

## 機能
- 「次の名言」で、名言と作者をランダムに表示
- 直前と同じ名言は出さない（連続で同じにならない）

## 使った技術
- Flutter / Dart（`Random`・`do-while` での選び直し・自作クラス Quote）
- 追加パッケージなし・**ネット通信なし**（名言はアプリ内に用意）

## 学び・ポイント
- 「前と同じを避ける」には、`do { 選ぶ } while (前と同じ)` で同じ番号を引いたら選び直す
- 候補が1つしかないときは無限ループにならないよう、先に `length <= 1` で抜ける
- 名言データはネットから取らず、アプリ内に持つと通信不要で軽い

## 注意
- このフォルダ単体ではビルドできません（上記 `flutter create .` が必要）。
