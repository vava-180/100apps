# Day 098: ミニ機械学習デモ（線形回帰）

「勉強時間からテストの点数を予想する」を題材に、いちばん基本の機械学習＝**線形回帰**（データに直線を当てはめる）を、ライブラリを使わず標準ライブラリだけで実装したデモです。

## 必要なもの
- Python 3（標準ライブラリのみ・追加インストール不要）

## 使い方
1. ターミナルでこのフォルダに移動する
2. `python linear_regression.py` を実行
3. 学習結果（直線の式・当てはまり具合）が表示される
4. 勉強時間を入力すると、点数を予想する（`q` で終了）

## 機能
- 教師データ（勉強時間と点数の組）から、最小二乗法で直線を学習
- 直線の式「点数 = 傾き × 勉強時間 + 切片」を表示
- 当てはまり具合 R^2（決定係数）を表示
- 入力した勉強時間から点数を予想（0〜100点に丸めて表示）
- 入力チェック（数字以外・NaN/無限大・負の値・q終了）

## 使った技術
- Python（標準ライブラリ math のみ）

## 学び・ポイント
- 線形回帰の傾きは `sum((x-平均x)(y-平均y)) / sum((x-平均x)^2)`、切片は `平均y - 傾き × 平均x`
- 決定係数 R^2 は `1 - 残差平方和 / 全体平方和`。全体平方和が0のときは別扱いにしないと誤解を招く
- 学習データも `math.isfinite()` で検証しておくと、NaN/無限大の混入で結果が壊れない
- ライブラリ関数は `ValueError` を投げ、CLI側で受けて短いメッセージにすると初心者にやさしい

## 注意
- 完全にローカルで動きます（外部APIやネット通信はありません）。
- これは学習用の最小デモです。実データの予測精度を保証するものではありません。
