# Day 002: 全角↔半角変換

英数字・記号・スペース・カタカナを、全角と半角でまとめて変換するブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. テキストを入力する
3. 変換したい種類（英数字／スペース／カタカナ）にチェックを入れる
4. 「半角にする」または「全角にする」を押す
5. 「コピー」で結果をコピーできる

## 機能
- 英数字・記号の全角⇔半角変換
- スペースの全角⇔半角変換
- カタカナの全角⇔半角変換（濁点「ガ」⇔「ｶﾞ」、半濁点「パ」⇔「ﾊﾟ」にも対応）
- 変換する種類をチェックボックスで選べる
- 結果のワンクリックコピー

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 半角と全角の英数字は文字コードがちょうど `0xFEE0` ずれているので、足し引きだけで変換できる
- カタカナの濁点・半濁点は「2文字（カナ＋゛）」を先に処理してから1文字ずつ変換するのがコツ
