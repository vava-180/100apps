# Day 035: 日報・議事録テンプレ自動生成ツール（Python）

「日報」「議事録」「週報」などの定型フォーマットを、日付入りで自動生成する実務ツールです（外部ライブラリなし・ネット通信なし）。
毎日ゼロから書き出す手間を減らし、書く項目（見出し）を先に用意します。

## 必要なもの
- Python（3.8以上の目安）

## 使い方
1. ターミナルで `python make_template.py`（WindowsならダブルクリックでもOK）
2. テンプレの種類を選ぶ（1=日報／2=議事録／3=週報）
3. 日付を入れる（空Enterなら今日。`2026-06-23` や `2026/06/23`）
4. 保存先フォルダを入れる
5. プレビューを確認し、`yes` で `2026-06-23_日報.md` のようなファイルを作成

## 機能
- 3種類のテンプレ（日報／議事録／週報）
- 日付・曜日・週の範囲を自動で埋め込み（例: `2026年6月23日(火)`）
- ファイル名は日付＋種類（並べると日付順にそろう）
- 既にあるファイルは**上書きしない**（連番を付けて新規作成）

## 使った技術
- Python（標準ライブラリ `datetime` / `pathlib` / `sys` のみ・外部通信なし）

## 学び・ポイント
- 曜日は `date.weekday()`（月曜=0）で出して、日本語の表に当てると `月〜日` に変換できる
- 週の範囲は「その日から月曜まで戻る」「月曜＋6日で日曜」で求められる。
  年をまたぐ週もあるので、**年を付けて**曖昧さをなくす
- 差し込みは `str.format` だと、テンプレ本文に普通の波括弧（`{メモ}` など）が
  混ざったときに壊れる。決めた語だけを `replace` で置き換えると、**テンプレを自由に編集しても安全**
- ファイルは `"x"`（新規作成専用）で開き、既にあれば連番にして上書き事故を防ぐ

## 注意
- 作るのはひな形（空欄つき）です。中身は作成後に書き込んでください。
- 文字コードは UTF-8 で保存します（Markdownとして開けます）。
