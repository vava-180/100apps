# Day 041: 名簿からシート一括作成（Excel VBA）

フェーズ3（VBA / Office自動化）の1本目。
名簿シートの **A列（2行目以降）** の値を、そのまま**シート名**にして、1行＝1枚のシートをまとめて作るExcelマクロです。

例: A2＝営業部, A3＝開発部, … → 「営業部」「開発部」…のシートが一気にできます。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. `sample/名簿サンプル.xlsx` を開く（または自分の名簿を用意。A1が見出し、A2以降に名前）
2. `Alt + F11` でVBE（マクロの画面）を開く
3. メニュー「ファイル → ファイルのインポート」で `CreateSheetsFromList.bas` を取り込む
   （または中身をコピーして、標準モジュールに貼り付け）
4. `Alt + F8` →「CreateSheetsFromList」を選んで「実行」
5. 確認のメッセージで「はい」を押すと、シートが一括で作られる

## 機能
- A列の値からシートを一括作成（実行前に確認のメッセージ）
- シート名の自動調整
  - 使えない文字 `\ / ? * [ ] :` は `_` に置き換え
  - タブ・改行・全角スペースなどは半角スペースにして前後を整える
  - 31文字を超える分は切り詰め
  - 予約名 `History` は `History_` に
  - 同じ名前は `(2)(3)…` を付けて重複回避
- 空欄の行・エラー値（`#N/A` など）の行は飛ばす
- 名前を付けられなかった場合は、できた空シートを残さない
- 画面更新の設定は、実行前の状態に戻す

## 使った技術
- Excel VBA（Worksheets.Add / On Error によるエラー処理）

## 学び・ポイント
- シート名にはExcelの制限（31文字・使えない記号・予約名・重複不可）がある。
  代入する前に名前を**整えてから**作ると、途中で止まらない
- シート追加と名前付けは**別関数**にして、失敗したら追加した空シートを消すと、
  ブックに余計なシートが残らない
- `ScreenUpdating` は実行前の値を**覚えておいて元に戻す**と、他の操作に影響しない
- `On Error GoTo` で、途中でエラーが出ても画面設定を戻してから知らせる

## 注意
- マクロを含むブックとして保存する場合は `.xlsm` 形式で保存してください。
- このマクロはシートを作るだけです（中身のコピーはしません）。
- 名簿は「いま開いているシート」を使います。名簿シートを表示してから実行してください。
