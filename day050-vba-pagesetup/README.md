# Day 050: 印刷範囲・ページ設定の一括適用（Excel VBA）

ブックの中のすべてのワークシートに、同じ印刷設定をまとめて適用するマクロです。
シートごとに手で設定する手間を省きます。

## 必要なもの
- Microsoft Excel（VBAが使えるデスクトップ版）

## 使い方
1. 設定したいブックを開く（`sample/印刷設定サンプル.xlsx` でも試せます）
2. `Alt + F11` でVBE →「ファイル → ファイルのインポート」で `ApplyPageSetup.bas` を取り込む
3. `Alt + F8` →「ApplyPageSetup」を実行
4. 用紙の向き（縦／横）と、横1ページに収めるか を選ぶ

## 機能
- 全ワークシートに一括適用（用紙の向き／横1ページ／印刷範囲／中央寄せ／ページ番号フッター）
- 「横1ページに収める」を no にすると、通常倍率（100%）に戻す
- 印刷範囲＝各シートの使用範囲。データの無いシートは印刷範囲を解除
- 保護シートは飛ばし、1シート失敗しても残りは続ける（失敗シート名を表示）
- `PrintCommunication` を使って高速化（Excel 2010以降）

## 使った技術
- Excel VBA（Worksheet.PageSetup / Application.PrintCommunication）

## 学び・ポイント
- 「横1ページに収める」は `Zoom = False` ＋ `FitToPagesWide = 1` の組み合わせ。
  `FitToPagesTall = False` にすると縦は成り行き（複数ページOK）になる
- フッターの `&P` は現在ページ、`&N` は総ページを表すExcelの記号
- PageSetupはプリンタと通信して遅いので、`PrintCommunication = False` でまとめて変更し、
  最後に `True` に戻すと速い（戻し忘れないよう、エラー時も必ず戻す）
- 1シートの失敗で全体を止めず、シート単位でエラーを集計すると使いやすい

## 注意
- 全ワークシートが対象です（グラフシートは対象外）。入力用・設定用シートも変わるので注意。
- プリンタ設定により、少し時間がかかることがあります。
- マクロを含むブックは `.xlsm` 形式で保存してください。

## サンプル
- `sample/印刷設定サンプル.xlsx` … 12か月分の売上表＋メモシート。
  実行すると、両シートに「横向き・横1ページ・ページ番号」などがまとめて設定されます。
