# Day 001: 文字数カウンター

入力したテキストの文字数・行数などをリアルタイムで集計するブラウザツールです。

## 使い方
1. `index.html` をブラウザで開く
2. テキストエリアに文章を入力する
3. 文字数・空白を除いた文字数・行数・原稿用紙換算が自動で表示される

## 機能
- 文字数のリアルタイム集計（絵文字も1文字として正しくカウント）
- 空白を除いた文字数
- 行数
- 原稿用紙換算（400字＝1枚）
- `-テキスト-` で囲んだ部分を引用として集計から除外

## 使った技術
- HTML / CSS / JavaScript（1ファイル完結・外部通信なし）

## 学び・ポイント
- 文字数は `text.length` ではなく `[...text].length`（スプレッド構文）で数えると、
  絵文字などの「サロゲートペア」も1文字として正しく数えられる
- `toLocaleString()` で数字を3桁区切り表示にできる
